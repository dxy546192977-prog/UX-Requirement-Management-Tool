# 平台通用组件

目录约定（与 `docs/component-index.md` 一致）：

- `widgets/` — 气泡、小红点、Switch、单选复选等小组件  
- `buttons/` — 常用按钮、浮动按钮、收起展开  
- `tags/` — 心智 / 服务 / 大促 / 优惠标签  
- `navigation/` — 导航栏、Tabbar、Tab  
- `feedback/` — 对话框、浮层、小黄条、Toast、空态  
- `selection/` — 时间、目的地、优惠券选择  
- `display/` — banner 等  

每个组件：`spec.md` + 可选 `example.html`。
