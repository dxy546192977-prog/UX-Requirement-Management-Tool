# vacation-search-category-nav · 类目导航

## 用途

搜索框下 **一级频道**：综合、跟团游、自由行、出境必备。

## 尺寸（@2x）

| 属性 | 值 |
|------|-----|
| 整体 | `750 × 106px` |
| 单项宽 | `188px`（四等分） |
| 背景 | 上灰半透 → 下白渐变 + `backdrop-filter: blur(10px)` |
| 顶圆角 | 承载层 `24px` |

## 态

- **选中**：白底凸起（高约 `106`）、字 `36px` `#0F131A`、icon 深色 + 黄渐变填充（与设计稿一致即可）。
- **未选中**：字 `30px` `#5C5F66`、icon 描边 `#5C5F66`。
