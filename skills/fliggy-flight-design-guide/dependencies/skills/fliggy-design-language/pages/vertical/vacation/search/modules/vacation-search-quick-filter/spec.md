# vacation-search-quick-filter · 快筛

## 用途

下拉筛下方的 **横向滚动** 二级标签（如：只看目的地参团、私家团…）。

## 尺寸（@2x）

| 属性 | 值 |
|------|-----|
| 栏高 | `72px` |
| 内边距 | 左 `18px`，上下 `6px` |
| chip 高 | `60px`，左右内边距 `18px`，圆角 `6px` |
| 间距 | chip 之间 `12px` |

## 态（与框架 6.4 一致）

- **默认**：底 `#F7F8FA`，字 `#0F131A`，`24px`。
- **选中**：底 **`#FFF2F0`**，字 **`#FF5533`**。

> 供稿 demo 若使用黄底选中，以本页框架为准。
