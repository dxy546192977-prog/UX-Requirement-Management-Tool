# 度假 · 门票/玩乐商品详情页 · 页面框架

## 画布与滚动

- **基准画布**：`750 × 1624`（@2x，iPhone 11 系），长页纵向滚动。
- **页面背景**：`#F2F3F5`。
- **主内容卡片区**：宽 `714px`，左右各 `18px` 边距；模块之间竖向间距 **`18px`**（6 的倍数）。
- **通栏媒体**：头图区宽 `750px`。

## 固定层

| 区域 | 行为 |
|------|------|
| **顶部吸顶栏** | `position: fixed; top: 0; z-index` 高于头图；**盖住头图**；宽 `750px`、高 **`176px`**（状态栏 `88` + 导航 `88`）。初始透明底、icon/字 **白色**；**滚动 `scrollY ≥ 180`** 时背景 `#FFFFFF`，icon/字 `#0F131A`。 |
| **底部 Bottom Bar** | `position: fixed; bottom: 0; left: 50%; transform: translateX(-50%);` 宽 `750px`、高 **`173px`**（含安全区）；白底、顶 `1px #D2D4D9`；**不随内容滚动**。 |

## 滚动内容顺序（自上而下）

1. 头图区 `750×750`（随页面上移，不吸顶）
2. 头图指示器（胶囊 tabs，宽 `714`、高 `72`，水平居中于 18 边距内）
3. 商详头部基础信息区（白卡 `714`）
4. 商详货架（白卡 `714`）
5. POI 信息区（白卡 `714`）
6. 图文详情（白卡 `714`）
7. **底部留白**：`padding-bottom` ≥ 底栏高度，避免内容被遮挡

## 文本与卡片

- **普通说明性 Markdown/正文**：铺在 `#F2F3F5` 上，**不**包进无定义的白卡。
- **白底圆角 `12px` 卡片**：仅用于框架约定的业务模块（基础信息、货架、POI、图文详情等）。
