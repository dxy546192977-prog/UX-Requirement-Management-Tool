# vacation-detail-shelf · 商详货架（套餐/SKU）

## 用途

票种/套餐售卖区：标题、日历说明入口、SKU 行（名称、副文案、标签、预订须知、价格、「订」按钮）。

## 容器（@2x）

| 属性 | 值 |
|------|-----|
| 宽 | `714px` |
| 背景 | `#FFFFFF` |
| 圆角 | `12px` |

## SKU 行

- 左右分栏：左信息区、右价格 + **订购按钮 `84×84`**，背景 `#FF7300`，圆角 `12px`，白字「订」。
- 价格主色 `#FF5533`；行分割线 `#F1F2F4`。
- 类目 tab（若存在）：选中 `#FFF7CC` / `#FFE033`，未选 `#F7F8FA`，圆角 `6px`。
