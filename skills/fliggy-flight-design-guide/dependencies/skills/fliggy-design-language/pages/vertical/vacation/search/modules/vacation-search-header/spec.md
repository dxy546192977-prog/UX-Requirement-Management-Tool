# vacation-search-header · 状态栏 + 顶搜

## 用途

列表页最上：**系统状态栏** + **返回 / 黄框搜索 / 更多**。

## 尺寸（@2x）

| 区块 | 高 |
|------|-----|
| 状态栏 | `88px` |
| 二级导航与搜索 | `100px` |
| **合计** | **`188px`**（计入整页吸顶 438 的前两段） |

## 搜索框

- 高 `72px`，圆角 `36px`，背景 `#FFFFFF`，边框 **`3px solid #FFE033`**。
- 内左右 `20px`；放大镜 `32×32`；关键词 `24px` `#0F131A`。
- 左右按钮 `48×48`。
