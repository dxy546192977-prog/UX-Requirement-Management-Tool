# 度假 · 旅行列表聚合页（小搜）· 页面框架

## 画布

- **基准**：`750 × 1624`（@2x），列表型页面。
- **页面背景**：`#F2F3F5`（固定吸顶区下方列表容器为白底时可 `#FFFFFF`）。

## 滚动模型（强约束）

**整页文档流不承载主列表滚动**。仅 **跟团游卡 + POI 卡** 所在区域为独立滚动容器：

| 层级 | 行为 |
|------|------|
| 状态栏 + 搜索行 + 类目 + 下拉筛 + 快筛 | **固定叠在最上层**（`position: fixed; top` 递增或单容器包起），不随列表滚动 |
| 混合列表 | `position: fixed; left: 50%; transform: translateX(-50%); width: 750px; top: 438px; bottom: 0; overflow-y: auto;`，**裁切** `overflow`，内容不可透出到吸顶区后 |

## 吸顶区高度（合计 438px）

| 区块 | 高度 |
|------|------|
| 状态栏 | `88px` |
| 搜索行（返回 + 搜索框 + 更多） | `100px` |
| 类目导航 | `106px` |
| 下拉筛选 | `72px` |
| 快筛 | `72px` |
| **合计** | **`438px`** → 列表 `top: 438px` |

## 列表区

- 宽 `750px`，与画布对齐。
- 卡片 **无缝竖排**，**item_gap: 0**（无额外模块间距、无分割线）。
- 支持 **跟团游卡** 与 **POI 卡** 混排。

## 文本

普通说明性 Markdown 铺在 `#F2F3F5`；**禁止**无定义白卡包裹正文。白底仅用于业务模块（搜索承载、筛选条、卡片等）。
