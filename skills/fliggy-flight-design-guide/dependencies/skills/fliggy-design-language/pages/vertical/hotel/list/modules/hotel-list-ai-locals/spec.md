# 酒店列表 — AI 本地人认证入口（`hotel-list-ai-locals`）

## 用途

**含 AI 快筛** 变体：紫粉渐变页背景；状态栏 + 顶栏（**攻略** / **地图** + 白圆搜索条）+ 指向箭头图；**本地人认证** 圆角内容卡（横滑目的地 + 图文推荐）；底部 **白底双行筛**（与常规主/快筛一致）。可与列表流上下拼接，避免与 `hotel-list-top-chrome` 重复叠两层主筛。

## 设计基准

- `viewport` **`width=750, user-scalable=no`**  
- 高亮城市 **#6666FF**；内容卡渐变 **#EBECFA → #FFFFFF**
