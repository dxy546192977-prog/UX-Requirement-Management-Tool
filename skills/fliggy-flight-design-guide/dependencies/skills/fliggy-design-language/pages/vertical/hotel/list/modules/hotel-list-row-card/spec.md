# 酒店列表 — 行卡（列表模式）（`hotel-list-row-card`）

## 用途

**750×312** 白底行卡：**左 222×288** 图区（角标、播放、底品质条）+ **右** 标题/豪华/金牌、蓝分、位置、标签、热卖、预订量与价格、渐变优惠条；底 **474px** 分割线。

## 设计基准

- 评分 **#6666FF** 底白字；价格 **#FF5533**；`SF Pro Text` 用于金额（`local` 降级）  
- `viewport` **`width=750, user-scalable=no`**
