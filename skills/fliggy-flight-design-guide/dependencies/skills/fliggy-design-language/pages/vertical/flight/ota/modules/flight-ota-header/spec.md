## 用途

OTA 结果页 **固定顶栏**：状态栏占位 + 返回 + **居中航线组**（城市 + **单程 icon** + 城市）+ 右侧 **低价提醒 / 我的收藏 / 更多**。

---

## 规格摘要

| 项 | 值 |
|----|-----|
| 总高 | **176px**（**88** + **88**） |
| 中线 | 航线组 **绝对居中**，禁止因左右宽度导致偏移 |
| 禁止 | 用 `—`、`->`、`→` 替代 **中间 icon** |
| 资产 | 返回 **48×48**；单程 icon **42×42**；提醒/收藏 **36×36**；更多 **48×48**；状态栏右 **134×24** |
