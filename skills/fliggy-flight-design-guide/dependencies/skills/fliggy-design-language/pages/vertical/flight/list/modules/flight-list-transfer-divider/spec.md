## 用途

直飞列表与中转列表之间的 **视觉分割**：灰底 + 左右 **220×1** 分割线切图 + 中间文案（如 **「以下为其他中转航班」**）+ 上下边界线。

---

## 规格摘要

| 项 | 值 |
|----|-----|
| 容器 | **750×90px**，背景 **#F2F3F5** |
| 文案 | **24px**，色 **#5B5F67**，居中 |
| 分割线 | 左右切图 **flex: 1**，透明度 **0.5**（供稿 URL 见 `example.html`） |

详参供稿 `输出组件/分割/README.md`。
