# 机票首页 — 小搜模块（`flight-home-mini-search`）

## 元信息

- **模块 slug**：`flight-home-mini-search`  
- **所属页**：`vertical.flight.home`  
- **示例**：[`example.html`](example.html)  
- **依赖**：[`foundations/design-foundations.md`](../../../../../../foundations/design-foundations.md)  
- **最后同步**：2026-04-05  

---

## 用途

机票首页 **核心搜索区**：一级交通 Tab（机票 / 特价机票 / …）+ 二级 Tab + 白底搜索卡片（OD、日期、乘机人、舱位、黄钮、历史）。

---

## 与框架文档的差异

- **框架文案** 中二级 Tab 为「单程 / 往返 / 多程」。  
- **当前 `example.html`** 以实现稿为准，为「**国内 / 国际 / 港澳台**」。若产品改回行程类型，仅更新本模块示例与 copy，**不改变** `page_id` / slug。  

---

## 规格摘要（摘录）

| 区块 | 要点 |
|------|------|
| 一级 Tab 区 | 毛玻璃 + 黄渐变指示条 **84×24**；切图与 SVG 见示例 |
| 二级 Tab | 选中 **30px/500/#0F131A**，下划线约 **46×6** |
| OD 行 | 城市 **42px/500**，换向图标 **68×68** |
| 日期 | 数字 **Fliggy Sans 102 48px**；「月」「周几」**26px PingFang** |
| 乘机人 | 标题 **30px/500**；提示 **25px/#149AA8**；可选气泡 |
| 舱位 | 胶囊约 **323px** 宽 |
| 搜索按钮 | **#FFE033**，高 **84**，圆角 **42**，文案 **32px/500** |
| 促销角标 | 红渐变、不对称圆角（见示例） |
| 历史 | **22px/#919499** |

**字体**：示例内 **Fliggy Sans 102** 使用 CDN（与 FDL 其他页一致），非本地 `Fonts/` 路径。
