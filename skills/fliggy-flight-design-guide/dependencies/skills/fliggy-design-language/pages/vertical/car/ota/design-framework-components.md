# 租车 OTA · 设计框架与组件来源

生成或校对 **整页 OTA** 时以本文件与 `manifest.md` 为准：严格复用本仓库 `modules/` 下已有切片结构，**只允许**替换文案、图片、显隐与局部组合，**不要**自造新卡片结构。

## 1. 页面骨架（固定）

```text
第 1 层：头图模块
第 2 层：车型摘要区（车辆信息 + 88VIP 黄条 + 取还车信息）
第 3 层：报价列表区（8 张报价卡 + 展开条 + 分组标题 + 推荐车型卡）
第 4 层：底部安全区 / Home Indicator
```

固定顺序：

```text
头图 -> 车型信息 -> 88VIP 黄条 -> 取还车信息 -> 报价列表 -> Home Indicator
```

## 3. 报价卡变体（与列表 8 张对应）

1. **品牌周·精选车**：基础报价卡，无 Tab。
2. **总价最低·特惠车**：双 Tab，左侧激活。
3. **品质新车**：双 Tab 左激活 + 商家行前 **`飞猪优选`** 徽章。
4. **飞猪优选车（一）**：无 Tab；`2年内车龄 / 自行到店 / 可选收费上门送取车`（第三行灰色）。
5. **飞猪优选车（二）**：无 Tab；`2年内车龄` + `取·…` + `还·…` 三行。
6. **全国连锁·大牌好车**：双 Tab 左激活 + **`全国连锁`** 徽章。
7. **小伤免赔·剐蹭无忧车**：标题右侧 **18×18** 箭头。
8. **一口价·舒心租车**：标题金棕色 **#805540**，底部 **6 项权益栏**。

## 4. 强识别与显隐

- 头图：顶栏仅 **左返回、右更多**；底栏 **视频 / VR / 外观 / 前排 / 后排 / 相册** + 右侧提示条 + 进度条（以设计稿为准）。
- 黄条 OTA 页：**88VIP** 文案 `专项优惠 ¥ 200 新能源车可享6折`（可改数字，结构不变）。
- 报价 CTA：**84×84** 橙色 **订** 按钮。
- 仅展示稿面需要的徽章：**飞猪优选**、**全国连锁**；勿额外加隐藏态元素。

## 5. 优先级（冲突时）

```text
最终设计截图 > 本框架文档 > 单组件 demo / Figma 隐藏分支
```
