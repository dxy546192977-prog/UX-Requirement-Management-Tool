# 安心租首页 — 设计框架与组件规范

与 [`page-frame.md`](page-frame.md) 配套：整页拼装、模块顺序、Token、高保真校验。生成完整页时 **先读本文件 + `page-frame.md`**。

---

## 一、页面定位

- **安心租 / 租车频道首页**：暖金头图 + 大白搜索卡 + 轻营销 + 白底列表 + 频道底栏。  
- **画布**：**750px** 宽；整页 **`min-height: 2763px`**（列表可增高）。  
- **主内容流**：**702px** 宽，与头图 **压接**（`margin-top: -41px`）。

---

## 二、五层骨架（禁止打乱）

```text
1. 顶部氛围（底层）
2. 搜索模块（压接头图，第一交互层）
3. 营销模块（轻量承接）
4. 列表模块（统一白底大容器）
5. 底部导航（贴底，不嵌进列表白底）
```

固定顺序：**顶部氛围 → 搜索 → 营销 → 列表 → 底栏**。

---

## 三、组合规则（摘要）

1. **顶部氛围**：独立沉浸式层，暖色，**非**「导航条 + 单图」拼凑；切图优先整图 750×267。  
2. **搜索模块**：单一大白卡；内部顺序固定 **2-1 切换 → 2-2 Tab → 2-3 国内/国际 → 2-4 营销条 → 2-5 地点 → 2-6 时间 → 2-7 筛选 → 2-8 按钮**；禁止把地点/时间/快筛拆成多张独立卡。  
   - 2-1 左侧未选：**机票 + 特价机票 / 火车票 / 汽车票**；右侧选中：**租车/接送**（演示态可强调租车场景）。  
   - 2-2 四个 Tab：**租车 / 接送机 / 接送火车 / 打车**。  
3. **营销模块**：更短、更轻，红色情绪 + 券轨，不做内容种草主舞台。  
4. **列表模块**：整块白底圆角容器；内：**标题头 → 城市 Tab → 4-3 视频组 → 车辆卡列表**。  
5. **底栏**：安静收尾，自带安全区示意；与主内容分层清晰。

## 六、整页容器模板（类名）

- **`.az-page`** — 750 宽、灰底、`min-height`、底栏占位  
- **`.az-hero-slot`** / **`.az-hero`** — 头图区  
- **`.az-main-stack`** — 702 宽、负 margin 压接、`gap: 18px`、`padding-bottom` 避开底栏  
- **`.az-bottom-slot`** — absolute 底栏  

可复制结构见 [`example-full.html`](example-full.html)。

## 七、整页校验清单

- [ ] 首屏先看到 **暖金头图**，而非大白卡单独占满  
- [ ] 搜索卡 **压** 在头图上，不断层  
- [ ] **黄钮「搜索租车」** 为最强 CTA  
- [ ] 营销区 **轻**，不抢搜索与列表  
- [ ] 列表 **收进白底大容器**，不散落在灰底上  
- [ ] 价格 **`var(--az-pay)`**、活动红、激活黄与稿一致  
- [ ] 底栏 **贴底**、安静，不与列表白底混叠
