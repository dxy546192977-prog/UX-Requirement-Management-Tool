## 模块

| slug | 说明 |
|------|------|
| `train-ota-title` | 状态栏 + 导航（日期 ▼ + 分享/更多） |
| `train-ota-itinerary` | 行程卡（OD、历时、车次、标签行） |
| `train-ota-promo-bar` | 营销条（品牌 icon + 文案 + 按钮） |
| `train-ota-seat-section` | 座席 Tab（折扣、价、余票、选中勾） |
| `train-ota-quote-section` | 报价方案（权益 / 官方 / 免登等） |
| `train-ota-watermark` | 飞猪水印整图 |
