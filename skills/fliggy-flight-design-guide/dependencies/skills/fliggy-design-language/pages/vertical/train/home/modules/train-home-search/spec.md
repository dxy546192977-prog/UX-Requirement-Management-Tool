# 火车票首页 — 搜索模块（`train-home-search`）

## 元信息

- **模块 slug**：`train-home-search`  
- **所属页**：`vertical.train.home`  
- **示例**：[`example.html`](example.html)  
- **依赖**：[`foundations/design-foundations.md`](../../../../../../foundations/design-foundations.md)  
- **最后同步**：2026-04-06  

---

## 用途

火车票首页 **核心搜索区**：一级五品类 Tab（机票 / 特价机票 / **火车票选中** / 汽车票 / 租车）+ 二级 **单程 / 往返** + 粉色优惠条 + 白底搜索卡（OD、日期 **Fliggy Sans 102**、学生票/高铁动车筛选、黄钮、角标、历史）。

---

## 规格摘要

| 区块 | 要点 |
|------|------|
| 整体 `.search-module` | **750×702**（上 **260** + 下 **442**） |
| 一级 Tab | 毛玻璃顶、选中白底切图、黄渐变条 **124×24**；品类切图位置见示例 |
| 二级 Tab | **702×98**，文案 **单程/往返**；选中 **30px/500/#0F131A**，下划线 **46×6**；第三 Tab 默认隐藏 |
| 优惠条 | **654×72**，`#FFF0F0`，圆角 **12**；文案含红色高亮「至¥68」等 |
| OD 行 | **100px**；城市 **42px/500**；换向 **68×68** 切图 |
| 日期行 | 数字 **Fliggy Sans 102 48px**；月日/星期 **26px PingFang** |
| 筛选行 | 复选 **28×28** 圆描边；文案 **26px/400/#5C5F66** |
| 搜索按钮 | **#FFE033**，**84** 高，圆角 **42**，「搜索火车票」**32px/500** |
| 促销角标 | 红渐变、不规则圆角（见示例） |
| 历史 | **22px/#919499** |

**字体**：示例内 **Fliggy Sans 102** 使用 CDN `@font-face`（与 FDL 其他页一致）。

**布局**：`.secondary-tab-wrapper` 为二级 Tab 外层容器，与 `tab-section-inner` 内顺序一致（见 `example.html`）。
