# 常见问题模块

对应设计稿 `6 常见问题`。与 `5 用研` 分离，只负责展示标题和问题胶囊组。

**AI 可配置项**：修改标题 / 配置 2~4 个问题胶囊 / 控制每个胶囊宽度 / 调整两行排列

## 规范参数（@2x）

### 外层容器 `.common-faq-block`

| 属性 | 值 |
|------|------|
| 宽度 | 750px |
| 参考高度 | 246px |
| 背景 | var(--color-white) |

### 标题头 `.common-faq-head`

| 属性 | 值 |
|------|------|
| 高度 | 80px |
| 内边距 | 24px |
| 标题字号 | 28px / 500 |

### 胶囊区 `.faq-chip-grid`

| 属性 | 值 |
|------|------|
| 内边距 | `0 24px 24px` |
| 行间距 | 18px |
| 列间距 | 20px |
| 推荐布局 | 两列 |

### 单个胶囊 `.faq-chip`

| 属性 | 值 |
|------|------|
| 最小高度 | 62px |
| 内边距 | `16px 42px` |
| 背景色 | var(--color-surface-muted) |
| 圆角 | 30px |
| 图标 | 24 x 24px |
| 文字字号 | 26px |
| 文字颜色 | var(--color-midgray) |

完整 HTML 结构详见 [example.html](example.html)。
