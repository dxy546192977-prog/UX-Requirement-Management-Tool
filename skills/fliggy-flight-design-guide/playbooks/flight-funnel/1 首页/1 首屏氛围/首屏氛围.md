# 首屏氛围 (TopBar)

## 这是什么？

这是机票首页最顶部的"氛围区"模块。你可以把它想象成一张全屏的品牌海报，覆盖在页面最上方，营造沉浸式的视觉氛围。

它由以下几个部分组成：
*   **背景图**：一张 750×310px 的全宽图片，支持活动 Banner、品牌氛围或季节主题等多种素材。
*   **状态栏占位**：顶部 88px 的空白区域，留给手机系统状态栏（时间、信号、电量等），不需要手动绘制。
*   **导航栏**：包含左侧的返回按钮和右侧的相机、更多两个操作按钮，都是 60×60px 的毛玻璃圆形按钮。
*   **轮播指示器**：右下角的小圆点，表示当前是第几张轮播图。
*   **底部渐变**：一个 85px 高的渐变遮罩，让背景图和下方的灰色页面背景（`#f2f3f5`）自然过渡，避免硬切边。

## 如何配置它？

### 背景图

*   **图片地址 (`backgroundImage`)**: 背景图通过 CDN 地址引用，建议使用 1500×620px（@2x）的图片，展示区域为 750×310px，使用 `cover` 模式填充。
    *   例如：`backgroundImage: 'https://gw.alicdn.com/imgextra/...png'`

*   当前预设的背景图：
    *   `https://gw.alicdn.com/imgextra/i2/O1CN01WZ8eNv1wPuk5smgbE_!!6000000006301-2-tps-1500-620.png`

### 导航栏按钮

导航栏有 3 个按钮，都是 60×60px 的圆形毛玻璃按钮（半透明黑底 `rgba(0,0,0,0.2)` + `blur(10px)`），内部图标为 36×36px 的白色 SVG。

*   **返回按钮 (`backAction`)**: 左侧的返回箭头，点击后返回上一页。
    *   例如：`backAction: { type: 'event', name: 'onBack' }`

*   **相机按钮 (`cameraAction`)**: 右侧第一个按钮，通常用于扫一扫或拍照搜索。
    *   例如：`cameraAction: { type: 'event', name: 'onCamera' }`

*   **更多按钮 (`moreAction`)**: 右侧第二个按钮（三个点），通常用于打开分享菜单。
    *   例如：`moreAction: { type: 'event', name: 'onMore' }`

你可以按产品需求增删右侧按钮，只需在 `.nav-bar__actions` 容器内添加或移除 `.nav-bar__action-btn` 即可。

### 轮播指示器

*   **轮播页数 (`slideCount`)**: 指示器圆点的数量，对应轮播图的总页数。
    *   例如：`slideCount: 4`

*   **当前页 (`activeIndex`)**: 当前选中的轮播页，从 0 开始。选中态圆点宽 18px（白色），未选中态宽 12px（40% 透明白色）。
    *   例如：`activeIndex: 0`

当前预设为 4 个圆点，第 1 个为选中态。

## 一些例子

### 1. 替换背景图

只需修改 `.top-bar__bg` 的背景图 URL：

```css
.top-bar__bg {
  background: url("你的新图片CDN地址") center top / cover no-repeat;
}
```

### 2. 修改轮播指示器数量

在 `.top-bar__slider` 容器内增减圆点，用 `--active` 类标记当前页：

```html
<div class="top-bar__slider">
  <span class="top-bar__dot top-bar__dot--active"></span>
  <span class="top-bar__dot"></span>
  <span class="top-bar__dot"></span>
</div>
```

### 3. 添加一个新的导航栏按钮

在 `.nav-bar__actions` 容器内添加一个新按钮：

```html
<div class="nav-bar__action-btn">
  <svg class="nav-bar__action-icon" width="36" height="36" viewBox="0 0 36 36" fill="none">
    <!-- 你的图标 SVG path -->
  </svg>
</div>
```

## 设计细节备忘

*   **毛玻璃效果**：导航栏按钮使用 `backdrop-filter: blur(10px)` 实现毛玻璃效果，需要同时添加 `-webkit-backdrop-filter` 兼容 Safari。
*   **底部渐变**：渐变从顶部完全透明 → 70% 处 80% 不透明 → 底部纯灰色（`#f2f3f5`），确保与页面背景无缝衔接。
*   **z-index 层级**：状态栏和导航栏的 z-index 为 10，底部渐变为 2，确保按钮始终在背景图之上。
*   **状态栏**：88px 的状态栏区域仅做占位，实际内容由宿主 App 或 WebView 渲染。
