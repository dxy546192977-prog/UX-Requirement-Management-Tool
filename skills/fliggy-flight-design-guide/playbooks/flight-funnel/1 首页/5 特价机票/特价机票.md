# 特价机票 (DealFlights)

## 这是什么？

这是机票首页的"特价机票"推荐模块。你可以把它想象成一个完整的低价航班推荐卡片，用户可以在这里快速浏览不同主题的特价航班信息，并找到最便宜的机票。

它由以下几个部分组成：

* **标题栏**：显示模块标题"特价机票"、出发城市选择器和"查看更多"入口。
* **主题一级 Tab**：横向可滚动的主题标签，如"低价任意飞"、"猜你喜欢"、"航司特价"等，选中态有白色卡片 + 底部三角箭头指示器。
* **国内/国际二级 Tab**：一个带异形选中态的切换栏，显示"国内最低 ¥99起"和"国际最低 ¥499起"。
* **筛选行**：单程/往返切换 + 日期下拉选择。
* **航班列表**：展示 5 条特价航班，每条包含城市风景图、航线、日期、价格和优惠标签，前 3 条还有排行角标。
* **底部按钮**："更多低价方案"的行动按钮。

整个模块采用淡紫色渐变背景（`#f5f5ff` → `#f8f8ff`），视觉上非常清新，能有效吸引用户点击浏览。

## 如何配置它？

### 标题栏

* **模块标题 (`title`)**: 默认为"特价机票"，一般不需要修改。
* **出发城市 (`departCity`)**: 标题右侧的城市选择器，显示当前出发城市。

  * 例如：`departCity: '杭州出发'`
* **更多入口 (`moreAction`)**: 右侧"查看更多"的跳转行为。

  * 例如：`moreAction: { type: 'url', value: '/flight/deals' }`

### 主题一级 Tab

你可以配置多个主题 Tab，每个 Tab 包含图标和文字。Tab 支持横向滚动，数量不限。

* **Tab 名称 (`label`)**: Tab 上显示的文字，建议不超过 6 个字。

  * 例如：`label: '低价任意飞'`
* **Tab 图标 (`icon`)**: 一个 36x36px 的图标，通过 CDN 地址引用。

  * 例如：`icon: 'https://gw.alicdn.com/imgextra/...png'`
* **选中状态 (`active`)**: 标记当前选中的 Tab。选中态会显示白色卡片背景和底部三角箭头指示器。

  * 例如：`active: true`

### 国内/国际二级 Tab

这个区域使用切图实现，包含一个半透明渐变背景层和一个带投影的选中态异形白色块。

* **国内最低价 (`domesticPrice`)**: 国内 Tab 上显示的最低价格。

  * 例如：`domesticPrice: '¥99起'`
* **国际最低价 (`internationalPrice`)**: 国际 Tab 上显示的最低价格。

  * 例如：`internationalPrice: '¥499起'`

### 筛选行

* **行程类型 (`tripType`)**: 单程或往返，默认选中"单程"。

  * 可选值：`'单程'` | `'往返'`
* **日期筛选 (`dateFilter`)**: 日期下拉的显示文案。

  * 例如：`dateFilter: '近一周出发'`

### 航班列表

每条航班包含以下可配置项：

* **出发城市 (`from`)**: 航班出发城市。

  * 例如：`from: '杭州'`
* **到达城市 (`to`)**: 航班到达城市。

  * 例如：`to: '青岛'`
* **出发日期 (`date`)**: 航班日期，格式为"MM.DD 周X"。

  * 例如：`date: '08.02 周三'`
* **价格 (`price`)**: 航班价格，纯数字，前面会自动加上低价 icon。

  * 例如：`price: 109`
* **优惠标签 (`tag`)**: 价格下方的标签文案，建议不超过 4 个字。

  * 例如：`tag: '30天低价'`
* **城市风景图 (`image`)**: 108x80px 的城市风景图，通过 CDN 地址引用。

  * 例如：`image: 'https://gw.alicdn.com/imgextra/...png'`
* **排行 (`rank`)**: 前 3 条航班会显示排行角标（TOP1/TOP2/TOP3），第 4 条及之后不显示。

  * 可选值：`1` | `2` | `3` | `null`

### 底部按钮

* **按钮文案 (`buttonText`)**: 底部按钮的文字。

  * 例如：`buttonText: '更多低价方案'`
* **点击行为 (`buttonAction`)**: 按钮的跳转行为。

  * 例如：`buttonAction: { type: 'url', value: '/flight/cheap-list' }`

## 一些例子

### 1. 修改出发城市为"上海出发"

只需要修改标题栏中的出发城市文案：

```html
<span class="deal-flights__dropdown-text">上海出发</span>
```

### 2. 添加一个新的主题 Tab

在 `.deal-flights__tabs` 容器内添加一个新的未选中态 Tab：

```html
<div class="deal-flights__tab">
  <span class="deal-flights__tab-icon">
    <img src="你的图标CDN地址" alt="新主题" />
  </span>
  <span class="deal-flights__tab-label">新主题名称</span>
</div>
```

### 3. 修改航班数据

替换航班卡片中的城市、日期和价格即可。注意前 3 条航班有排行角标，第 4 条及之后没有：

```html
<div class="deal-flights__flight-item">
  <!-- 如果需要排行角标，加上这段 -->
  <div class="deal-flights__rank-badge">
    <img class="deal-flights__rank-badge-bg deal-flights__rank-badge-bg--1" alt="排行第1" />
  </div>
  <div class="deal-flights__flight-left">
    <div class="deal-flights__flight-img">
      <img src="城市风景图CDN地址" alt="目的地" />
    </div>
    <div class="deal-flights__flight-info">
      <div class="deal-flights__flight-od">
        <span class="deal-flights__flight-city">出发城市</span>
        <span class="deal-flights__flight-arrow-icon">
          <img src="https://gw.alicdn.com/imgextra/i1/O1CN016dWVXr25ZNT4wCSgT_!!6000000007540-55-tps-24-24.svg" alt="→" />
        </span>
        <span class="deal-flights__flight-city">到达城市</span>
      </div>
      <span class="deal-flights__flight-date">MM.DD 周X</span>
    </div>
  </div>
  <div class="deal-flights__flight-right">
    <div class="deal-flights__flight-price-row">
      <span class="deal-flights__price-icon">
        <img src="https://gw.alicdn.com/imgextra/i4/O1CN01UCcXiX1d32G3WOIke_!!6000000003679-2-tps-72-72.png" alt="¥" />
      </span>
      <span class="deal-flights__flight-amount">价格数字</span>
    </div>
    <div class="deal-flights__flight-tag">
      <span class="deal-flights__flight-tag-text">标签文案</span>
    </div>
  </div>
</div>
```

### 4. 切换二级 Tab 的选中态

二级 Tab 使用切图实现。如果需要切换到"国际"选中态，需要替换选中态背景 SVG 的位置和文字样式。

## 设计细节备忘

* **Tab 文字倾斜**：一级 Tab 的文字使用方正隽黑字体，并通过 `transform: skewX(-8deg)` 实现 8 度倾斜效果，注意不要使用 Figma 导出的 `matrix` 变换，那会导致文字被拉宽。
* **选中态指示器**：选中 Tab 底部的白色三角箭头通过 `clip-path: polygon(0 0, 100% 0, 50% 100%)` 实现。
* **二级 Tab 切图**：选中态的异形白色块使用独立 SVG 切图，背景层使用半透明渐变 PNG，两者叠加实现完整效果。
* **排行角标**：前 3 名使用不同颜色的角标切图（金/银/铜），通过 CSS `content` 属性按序号加载对应图片。
* **价格字体**：价格数字使用 Fliggy Sans 102 Bold 字体，确保数字显示风格统一。
