# 省钱模块 (SaveMoney)

## 这是什么？

这是机票首页的"出行省钱"推荐模块。你可以把它想象成一个横向排列的营销卡片墙，帮助用户快速发现优惠券、低价日历和特价机票等省钱入口。

它由以下几个部分组成：
*   **标题栏**：左侧显示模块标题"出行省钱"，右侧有"更多优惠"入口（带箭头图标）。
*   **卡片行**：横向排列 3 张营销卡片，每张卡片都有独立的背景图、标题、副标题和行动按钮。

三张卡片从左到右分别是：
1.  **红色折扣卡**（256px 宽）：机票优惠券，红色实心按钮"立即领取"。
2.  **金色推广卡-中**（242px 宽）：低价机票日历，带飞猪 F1 标签，金色描边按钮"去看看"。
3.  **金色推广卡-小**（156px 宽）：特价机票，金色描边按钮"去看看"。

整个模块是一个白色圆角卡片（702×256px，圆角 18px），视觉上简洁明快。

## 如何配置它？

### 标题栏

*   **模块标题 (`title`)**: 默认为"出行省钱"，一般不需要修改。

*   **更多入口 (`moreAction`)**: 右侧"更多优惠"的跳转行为。
    *   例如：`moreAction: { type: 'url', value: '/flight/deals' }`

### 卡片配置

每张卡片包含以下可配置项：

*   **卡片标题 (`cardTitle`)**: 卡片上方的主标题，26px/500。
    *   例如：`cardTitle: '机票优惠券'`

*   **卡片副标题 (`cardSubtitle`)**: 标题下方的说明文字，22px/400 灰色。
    *   例如：`cardSubtitle: '领券立减出行省'`

*   **背景图 (`backgroundImage`)**: 卡片的背景图片，通过 CDN 地址引用，使用 `cover` 模式填充。
    *   例如：`backgroundImage: 'https://gw.alicdn.com/imgextra/...png'`

*   **按钮文案 (`buttonText`)**: 底部行动按钮的文字。
    *   例如：`buttonText: '立即领取'`

*   **按钮样式 (`buttonStyle`)**: 按钮有两种样式可选。
    *   `'red'`：红色实心按钮（背景 `#ff6666`，白色文字，圆角 24px）
    *   `'gold'`：金色描边按钮（透明背景，`rgba(178,95,36,0.32)` 描边，金色文字 `#b25f24`，圆角 22px）

*   **F1 标签 (`showF1Tag`)**: 是否在标题前显示飞猪 F1 标签图标（34×24px）。
    *   例如：`showF1Tag: true`

当前预设的 3 张卡片：

| 卡片 | 宽度 | 标题 | 副标题 | 按钮 | F1 标签 |
|------|------|------|------|------|------|
| 红色折扣卡 | 256px | 机票优惠券 | 领券立减出行省 | 立即领取（红色） | 无 |
| 金色推广卡-中 | 242px | 低价机票日历 | 看看哪天飞最便宜 | 去看看（金色） | 有 |
| 金色推广卡-小 | 156px | 特价机票 | 低至1折起 | 去看看（金色） | 无 |

## 一些例子

### 1. 修改卡片的标题和副标题

直接替换卡片内的文字即可：

```html
<div class="card__info">
  <div class="card__title-row">
    <span class="card__title">新的标题</span>
  </div>
  <span class="card__subtitle">新的副标题</span>
</div>
```

### 2. 替换卡片背景图

修改对应卡片类的 CSS `background-image`：

```css
.card--discount {
  background-image: url("你的新背景图CDN地址");
}
```

### 3. 添加一张带 F1 标签的新卡片

在 `.save-money__cards` 容器内添加一张新卡片，标题前加上 F1 标签图标：

```html
<div class="card card--promo-mid" style="background-image: url('背景图CDN地址');">
  <div class="card__info">
    <div class="card__title-row">
      <span class="card__tag">
        <img src="https://gw.alicdn.com/imgextra/i1/O1CN01ckRuPS233eo0NPAco_!!6000000007200-2-tps-68-48.png" alt="F1" />
      </span>
      <span class="card__title">新卡片标题</span>
    </div>
    <span class="card__subtitle">新卡片副标题</span>
  </div>
  <button class="card__btn--gold">去看看</button>
</div>
```

### 4. 将按钮从金色改为红色

将按钮的 class 从 `card__btn--gold` 改为 `card__btn--red`，并修改文案：

```html
<button class="card__btn--red">立即领取</button>
```

## 设计细节备忘

*   **卡片宽度不等**：三张卡片宽度分别为 256px、242px、156px，加上两个 12px 间距，总宽度恰好为 654px（容器内容宽度）。
*   **背景图 cover**：所有卡片背景图使用 `background-size: cover` + `background-position: center`，确保不同尺寸的图片都能正确填充。
*   **按钮尺寸统一**：红色按钮和金色按钮尺寸相同（108×42px），保持视觉一致性。
*   **F1 标签**：飞猪 F1 标签图标尺寸为 34×24px，使用 `object-fit: contain` 确保不变形。
