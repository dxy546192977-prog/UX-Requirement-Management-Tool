#### 价格区规则

- `少量余票型` 的 `少量` 在价格第一行，位于删除线原价左侧，不是单独一行。
- `当日低价型` 多一行 `特定乘客价¥440`。
- `直飞低价型`、`跨天型`、`少量余票型` 的底部只有优惠条 + 舱位文案两行。

## 推荐结构

### 直飞低价型

1. 顶部标签
2. OD 行
3. 航司信息行
4. 价格区：删除线原价 + 当前价 / 优惠条 / 舱位文案

### 当日低价型

1. 顶部标签
2. OD 行
3. 航司信息行
4. 权益行
5. 价格区：删除线原价 + 当前价 / 优惠条 / 舱位文案 / 特定乘客价

### 返现金型

1. OD 行
2. 航司信息行
3. 权益行（含共享）
4. 价格区：删除线原价 + 当前价 / 优惠条 / 舱位文案

### 跨天型

1. OD 行（到达时间上方插入跨天日期）
2. 航司信息行
3. 价格区：删除线原价 + 当前价 / 优惠条 / 舱位文案

### 少量余票型

1. OD 行
2. 航司信息行
3. 价格区第一行：`少量 + 删除线原价 + 当前价`
4. 价格区第二、三行：优惠条 / 舱位文案
