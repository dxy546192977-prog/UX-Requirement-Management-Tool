# 国内机票下单页 - 框架组件

国内机票下单页的页面框架由以下固定组件构成，从上到下依次为：顶部渐变背景层、iOS状态栏、导航栏（步骤指示器）、出票安心胶囊、航班信息卡、乘机人模块（含联系电话）、保险模块、会员专享模块、报销凭证模块、百叶窗提示、安心票心智、底部吸底栏。

> **设计基准**：750 × auto（@2x），viewport width=750, user-scalable=no
