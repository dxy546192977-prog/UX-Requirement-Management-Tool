# 心智模块

用于页面底部的品牌心智收尾，展示“飞猪旅行·安心票”和三段安心文案。

## 用途

- 作为整页滚动内容的收尾
- 强化飞猪品牌和安心票心智

## 默认可见态

- 上方品牌切图：`飞猪旅行·安心票`
- 下方文案：`出票安心 · 出行安心 · 服务安心`

## 结构层级

```html
<footer class="mindset-module">
  <img class="mindset-brand">
  <div class="mindset-copy"></div>
</footer>
```

## 关键尺寸 / 字体 / 排版

- 模块高度：约 `186px`
- 品牌切图：`282 × 41`
- 文案：`24px`
- 文案两侧带轻量横线

## 可选态

- 只保留品牌切图
- 只保留文案

## 组合约束

- 必须位于整页滚动内容底部
- 不固定在视窗底部
