# 框架组件

生成完整订单详情页时**始终读取本文件**。本文件只描述固定框架与一级分区，不承载业务文案细节。AI 仅替换标注为 `<!-- AI替换 -->` 的模块内容，不打乱结构顺序。

---

## 1. 页面容器（订单详情）

整页为 750px 宽的长页面，`display: flex; flex-direction: column; gap: 18px; background: #F2F3F5;`。

### 规范参数

| 属性 | 值 |
|------|------|
| 页面宽度 | 750px |
| 参考高度 | 3111px |
| 主布局 | `flex-direction: column` |
| 一级区块间距 | 18px |
| 页面背景 | var(--color-page-bg) |

### 可复制 HTML

```html
<div class="order-detail-page">
  <!-- 1 履约区 -->
  <!-- 2 行程区 -->
  <!-- 3 出行信息 -->
  <!-- 4 订单信息 -->
  <!-- 5 用研 -->
  <!-- 6 常见问题 -->
  <!-- 7 客服入口 -->
</div>
```

## 2. 1 履约区（固定框架区）

履约区本身就是一个复合框架，由背景、导航栏、订单状态、顶部按钮、履约卡组成。生成完整页面时，这一整区必须最先落下。

### 2.1 `1-1 背景`

| 属性 | 值 |
|------|------|
| 区块宽度 | 750px |
| 区块高度 | 755px |
| 背景底色 | var(--color-page-bg) |
| 渐变高度 | 774px |
| 渐变 | `linear-gradient(180deg, #FFE036 0%, rgba(255,255,255,0.0001) 81.77%)` |

### 2.2 `1-2 导航栏`

| 属性 | 值 |
|------|------|
| 总高度 | 162px |
| 状态栏高度 | 80.75px |
| 导航主栏高度 | 81.25px 左右 |
| 左侧 | 返回箭头 |
| 中部 | 双 tab：`国内备份` / `订单详情` |
| 右侧 | 36px 客服图标 + 48px 更多图标 |

### 2.3 订单状态 / 按钮 / 履约卡的位置关系

履约区内部固定顺序如下：

1. 导航栏
2. 订单状态
3. 顶部按钮组
4. 履约卡组

不要把按钮挪到履约卡下面，也不要把订单状态并入导航栏。

### 可复制 HTML

```html
<section class="fulfillment-zone">
  <div class="fulfillment-bg-layer"></div>

  <!-- 1-2 导航栏 -->
  <header class="fulfillment-nav">
    <!-- AI替换：仅替换标题状态和 icon 呈现，不改层级 -->
  </header>

  <!-- 1-3 订单状态 -->
  <!-- AI替换：订单状态头部 -->

  <!-- 1-4 按钮 -->
  <!-- AI替换：快捷按钮组 -->

  <!-- 1-5 履约卡 -->
  <!-- AI替换：履约服务卡组 -->
```

## 3.5 海免商品推荐（条件展示模块）

当航班出发或到达城市为三亚/海口，且订单已支付、未退款时，在行程区与出行信息之间展示海南免税商品推荐 Banner。不满足条件时不渲染、不占位。

### 规范参数

| 属性 | 值 |
|------|------|
| 外层宽度 | 750px |
| 内边距 | 0 18px |
| Banner 可视宽度 | 714px |
| Banner 圆角 | 12px（`var(--radius-l)`） |
| 与上下模块间距 | 18px（由页面 `gap` 控制） |
| 透出条件 | 出发/到达含三亚或海口 & 已支付 & 未退款 |

### 可复制 HTML

```html
<!-- 2.5 海免商品推荐（条件展示） -->
<section class="duty-free-recommend"
         data-spm="Product recommendation"
         data-condition-cities="三亚,海口"
         data-condition-status="paid"
         data-condition-refund="none">
  <a class="duty-free-recommend__banner" href="...">
    <img src="运营图URL" alt="海南免税购物">
    <button class="duty-free-recommend__close" aria-label="关闭推荐"></button>
  </a>
</section>
```
