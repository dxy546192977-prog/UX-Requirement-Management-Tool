# 行程详情模块

订单详情页的核心信息模块，包含价格摘要栏、订单号复制、段程时间轴、舱位与机型信息，以及中转提示。

**AI 可配置项**：修改总额 / 修改订单号 / 修改退改签入口文案 / 配置单程或往返 / 配置直飞或中转 / 修改时间轴节点与机场名称 / 配置中转说明卡

## 模块结构

### 1. 价格摘要栏 `.fare-summary`

| 属性 | 值 |
|------|------|
| 宽度 | 750px |
| 背景色 | var(--color-white) |
| 上内边距 | 30px |
| 模块间距 | 18px |

#### 主要信息行

- 左侧：`总额 + 价格`
- 右侧：`退改签、行李额、产品说明`
- 金额字体：24px / 500，可与 `Fliggy Sans 102` 混用

### 2. 订单号复制行 `.order-copy-row`

订单号与复制动作同排展示，字号 24px。

### 3. 段程卡 `.segment-card`

白底卡片，承接日期、展开控制、时间轴与飞行信息。

#### 日期头 `.segment-head`

| 元素 | 样式 |
|------|------|
| 段程标签 | 48 x 28px，黄色底，20px / 500 |
| 日期 | 26px / 500 |
| 航变标签 | 1px 红色描边，20px / 400 |
| 展开按钮 | 24px / 500 + 12px 箭头 |

#### 本地时间提示 `.local-time-row`

左侧说明“航班起降时间均为当地时间”，右侧为英文机场名开关。

#### 时间轴 `.flight-timeline`

支持三类节点：

1. 起飞节点
2. 飞行信息条
3. 到达节点

如为中转场景，可在中间插入 `.transfer-card`。

#### 中转卡 `.transfer-card`

| 属性 | 值 |
|------|------|
| 宽度 | 569px |
| 高度 | 62px |
| 背景色 | var(--color-surface-muted) |
| 圆角 | var(--radius-l) |
| 内边距 | `16px 12px 16px 18px` |

完整 HTML 结构详见 [example.html](example.html)。
