# 机票首页 - 框架组件

机票首页的页面框架由以下固定组件构成，从上到下依次为：首屏氛围（沉浸式导航栏）、小搜模块（核心搜索区域）、小金刚（快捷功能入口）、省钱模块（出行省钱推荐）、特价机票（低价航班推荐）、活动长图（图片占位）、底部导航栏。

> **设计基准**：750 × 1624px（@2x），viewport width=750

---

## 1. 首屏氛围（沉浸式导航栏）

顶部固定 | 750×310px | 状态栏 88px + 导航栏 88px + 轮播指示器 + 底部渐变

- **背景图**：750×310px，CDN 可配置，使用 `cover` 模式填充，建议源图 1500×620px（@2x）
- **状态栏**：88px 高，仅做占位，由宿主 App 渲染
- **导航栏**：88px 高，左右 padding 24px
- **按钮**：60×60px 圆形毛玻璃按钮（`rgba(0,0,0,0.2)` + `blur(10px)`），内图标 36×36px 白色 SVG
- **返回按钮**：左侧 left:24px，垂直居中
- **右侧按钮组**：相机 + 更多，间距 12px，right:24px
- **轮播指示器**：right:57px, top:210px，圆点 12×6px（未选中）/ 18×6px（选中）
- **底部渐变**：85px 高，透明 → 80% 不透明 → `#f2f3f5`，与页面背景无缝衔接

详细规范见 → [1 首屏氛围/首屏氛围.md](1 首屏氛围/首屏氛围.md)

## 2. 小搜模块（核心搜索区域）

页面核心交互区 | 上下双部分设计

**上半部分（一级 Tab + 二级 Tab）：**
- 一级 Tab：702×84px，5 个品类（机票/特价机票/火车票/汽车票/接送机），机票为选中态，选中态下方有黄色渐变指示条（84×24px）
- 二级 Tab：行程类型"单程 / 往返 / 多程"，选中态 30px/500/#0F131A + 下划线 46×6px

**下半部分（搜索卡片）：**
- OD 城市选择行：出发/到达城市名 42px/500，飞机换向按钮 68×68px 切图
- 日期选择行：日期数字使用 Fliggy Sans 102 字体 48px，"月"和"周几" 26px PingFang SC
- 乘机人行：标题 30px/500 + 绿色提示文本 25px/400/#149AA8 + 引导气泡
- 舱位切换：经济舱/公务舱/头等舱胶囊切换器，323px 宽
- 搜索按钮：黄色按钮 #FFE033 高 84px 圆角 42px，"搜索机票" 32px/500
- 促销角标：不对称圆角红色渐变背景，20px/500/白色
- 搜索历史：22px/400/#919499

详细规范见 → [2 小搜模块/小搜模块.md](2 小搜模块/小搜模块.md)

## 5. 特价机票（低价航班推荐）

淡紫色渐变卡片 702px | 圆角 18px | 标题 + Tab + 二级 Tab + 筛选 + 航班列表 + 按钮

- **外容器**：background `linear-gradient(167.59deg, #f5f5ff→#f8f8ff)`，backdrop-filter: blur(4px)，padding 24px
- **标题栏**："特价机票" 32px/500 + 出发城市下拉 + "查看更多"
- **一级 Tab**：横向滚动，6 个主题 Tab（低价任意飞/猜你喜欢/航司特价/免签落地签/头等商务舱特惠/畅游全球），选中态白色卡片 + 底部三角箭头
- **二级 Tab**：国内/国际切换，切图实现，选中态带 drop-shadow
- **筛选行**：单程/往返切换 + 日期下拉
- **航班列表**：5 条航班，每条含城市风景图(108×80px) + 航线 + 日期 + 价格 + 标签，前 3 条有排行角标
  - 价格字体：**Fliggy Sans 102** Bold，42px，色 #FF5533
- **底部按钮**："更多低价方案"，紫色背景 #F5F5FF，文字 #6666FF，圆角 117px
