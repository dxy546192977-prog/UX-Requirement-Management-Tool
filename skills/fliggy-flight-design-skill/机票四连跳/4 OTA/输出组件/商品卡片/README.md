# 商品卡片

本目录维护页面中 4 张默认可见商品卡，统一采用新的价格列、权益行和 tab 结构。

## 用途

- 复用国内机票 OTA 结果页中的 4 张报价卡
- 基于最新 CSS 命名和真实切图资产输出对应卡片

## 共享骨架

```html
<article class="goods-card">
  <header class="goods-head"></header>
  <div class="goods-card-body">
    <aside class="goods-price-col">
      <div class="price-row">
        <div class="price-main"></div>
        <div class="strike"></div>
      </div>
      <div class="price-note"></div>
    </aside>
    <section class="goods-info"></section>
    <button class="goods-buy"></button>
  </div>
</article>
```

## 隐藏态

- 航司机型行默认全部隐藏，不在 4 张卡中展开
- 02 顶部提醒行默认隐藏
- 删除线价可按卡片场景显示或隐藏
- `礼 / 限 / 旅行套餐 / 返现金` 按当前 4 张卡的默认态组合

## 组合约束

- 商品卡只维护卡片本身，不承载 `otatab` 和 `更多报价入口`
- 4 张卡统一使用新的价格列结构，删除线价不再单独占一行
- 03 / 04 的顶部 tabs 统一使用 `外层灰槽 + 选中白面板` 结构
- 04 的返现标签必须保留 `返现金切图 + 虚线分隔 + 金额 + 小箭头` 四段结构，不能简化成“图 + 文本”
- 根目录整页 `example.html` 装配顺序保持 `01 → 02 → 03 → 04`
