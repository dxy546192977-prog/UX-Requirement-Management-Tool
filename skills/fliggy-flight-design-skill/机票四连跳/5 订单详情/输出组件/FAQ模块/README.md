# FAQ 模块

订单详情页底部支持模块，包含两段内容：数字问答条与常见问题胶囊。用于把客服与 FAQ 能力收口到页面尾部。

**AI 可配置项**：修改 FAQ 标题 / 配置 2 个问答引导标签 / 配置数字问答条的数字范围 / 修改常见问题胶囊文案 / 控制每行胶囊数量

## 模块结构

### 1. 数字问答区 `.faq-digits-section`

包含顶部渐变标题、两枚问答引导标签和一条黄色数字问答条。

#### 标题区

| 属性 | 值 |
|------|------|
| 高度 | 92px |
| 背景 | `linear-gradient(180deg, #FFFDF2 0%, #FFFFFF 108.15%)` |
| 内边距 | `36px 24px 24px` |
| 标题字号 | 28px / 500 |

#### 引导标签 `.faq-intro-chip`

| 属性 | 值 |
|------|------|
| 高度 | 32px |
| 图标 | 32 x 32px |
| 文字字号 | 26px |
| 颜色 | var(--color-lightgray) |

#### 数字问答条 `.digit-strip`

| 属性 | 值 |
|------|------|
| 宽度 | 702px |
| 高度 | 72px |
| 背景色 | #FFF8CC |
| 圆角 | `36px 2px 36px 36px` |
| 数字气泡 | 52 x 52px 白底圆，1px 黄边 |

### 2. 常见问题区 `.common-faq-section`

白底区块，标题在上，问题胶囊分两行排列。

#### 问题胶囊 `.faq-chip`

| 属性 | 值 |
|------|------|
| 高度 | 62px |
| 背景色 | var(--color-surface-muted) |
| 圆角 | 30px |
| 内边距 | `16px 42px` |
| 文字字号 | 26px |
| 文字颜色 | var(--color-midgray) |
