# 小金刚

## 这是什么？

这是机票首页搜索区域下方的"快捷功能入口"模块。你可以把它想象成一排小按钮，每个按钮由一个 60×60px 的彩色图标和一行文字标签组成，用户点击后可以快速跳转到对应的功能页面。

整个模块是一个白色圆角卡片（702×128px，圆角 18px），内部横向排列 5 个等间距的功能入口。结构非常简单，配置起来也很方便。

## 如何配置它？

### 功能入口

每个入口包含两个可配置项：

* **入口名称 (`label`)**: 图标下方的文字标签，建议不超过 4 个字。字号 24px，居中显示。

  * 例如：`label: '特价机票'`
* **入口图标 (`icon`)**: 一个 60×60px 的图标，通过 CDN 地址引用。源图尺寸为 120×120px（@2x）。

  * 例如：`icon: 'https://gw.alicdn.com/imgextra/...png'`
* **点击行为 (`action`)**: 点击入口后的跳转行为。

  * 例如：`action: { type: 'url', value: '/flight/cheap' }`

## 一些例子

### 1. 替换某个入口的图标和文字

直接修改对应 `.kong__item` 中的图标 src 和文字即可：

```html
<button class="kong__item" type="button">
  <img class="kong__icon" src="你的新图标CDN地址" alt="新功能" />
  <span class="kong__label">新功能</span>
</button>
```

### 2. 增加或减少入口数量

在 `.kong__row` 容器内添加或移除 `.kong__item`。入口之间使用 `justify-content: space-between` 自动等间距分布，所以增减数量后无需手动调整间距。

```html
<div class="kong__row">
  <button class="kong__item" type="button">
    <img class="kong__icon" src="图标1" alt="功能1" />
    <span class="kong__label">功能1</span>
  </button>
  <button class="kong__item" type="button">
    <img class="kong__icon" src="图标2" alt="功能2" />
    <span class="kong__label">功能2</span>
  </button>
  <!-- 可以继续添加更多入口 -->
</div>
```

## 设计细节备忘

* **入口使用 `<button>` 标签**：每个入口使用语义化的 `<button>` 标签而非 `<div>`，自带可点击和无障碍支持。
* **图标尺寸**：显示尺寸 60×60px，源图建议 120×120px（@2x），确保在高清屏上清晰。
* **文字不换行**：标签使用 `white-space: nowrap`，超长文字不会换行，所以建议控制在 4 个字以内。
* **容器高度**：容器高度 128px 由 padding（上 18px + 下 22px）+ 内容行高度（88px）撑开，不需要手动设置固定高度。
