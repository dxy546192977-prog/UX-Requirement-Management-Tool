# 框架组件

机票列表页的页面骨架，定义固定布局和模块排列顺序。根目录 `listing.png`、`listing css.txt` 是最终还原基准，根目录 `example.html` 是当前完整拼装稿。

## 1. 页面结构说明

- 设计基准：750 × 1624px（@2x）
- 导航栏：吸顶固定，高度 176px（状态栏88px + 导航栏88px）
- 底bar：吸底固定，高度 160px（按钮区96px + 安全区64px）
- 中间内容区：可滚动，需预留顶部 padding-top: 176px 和底部 padding-bottom: 160px
- 模块排列顺序：日历 → 营销条 → 快筛栏 → 直飞卡片列表 → 省钱推荐（可选）→ 分割 → 中转快筛栏 → 中转卡片列表 → 心智模块
- 整页预览时建议使用居中页面壳：`.page { width: 750px; margin: 0 auto; }`，顶部导航和底bar都用 `left: 50%; transform: translateX(-50%);` 固定

## 2. 页面骨架模板

提供完整的 HTML 骨架代码，包含：
- DOCTYPE、html、head（含 viewport meta、字体引入）
- `<style>` 标签中的完整 `:root` 变量声明（基础 token + listing 扩展 token + 圆角）
- body 结构：页面壳 `.page` + 导航栏容器(fixed top) + 内容区容器(scrollable) + 底bar容器(fixed bottom)
- 每个模块的占位注释

## 3. 模块间距规则

- 日历与营销条之间：无间距（紧贴）
- 营销条与快筛栏之间：无间距
- 快筛栏与直飞卡片之间：无间距
- 直飞卡片之间：通过底部边框 1px solid #F2F3F5 分隔
- 省钱推荐：可选模块；当前 `example.html` 的 `listing.png` 还原稿未插入该模块
- 分割区域：90px 高度，背景 #F2F3F5；使用左右两张 220×1 分割线切图 + 上下边界线
- 中转快筛栏：使用独立 `.transfer-filter-wrapper` 骨架，内层滚动内容最小宽度 854px，不能直接复用顶部快筛栏 DOM
- 中转卡片之间：通过底部边框分隔
- 心智模块：独立区域
