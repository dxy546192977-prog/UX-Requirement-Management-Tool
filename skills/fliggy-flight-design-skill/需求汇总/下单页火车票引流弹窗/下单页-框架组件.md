# 框架组件（下单页 - 火车票引流弹窗扩展）

生成包含火车票引流弹窗的下单页时**始终读取本文件**。本文件在原有下单页框架基础上，补充弹窗层的框架描述。

---

## 1. 弹窗层级关系

火车票引流弹窗是一个**浮层组件**，覆盖在下单页之上，不影响下单页原有的 DOM 结构和布局。

### 层级规范

> **FDG 组件基线**：弹窗基于 `fdl-dialog`（`components/platform/feedback/dialog/spec.md`），z-index 遵循 FDG 标准值 1000。

| 层级 | 组件 | z-index |
|------|------|---------|
| 底层 | 下单页主体内容 | 0~10 |
| 中层 | 底部吸底栏 | 100 |
| 顶层 | 火车票引流弹窗（`fdl-dialog`） | 1000（FDG 标准） |

### 弹窗与下单页的关系

- 弹窗展示时，下单页内容**不可滚动**（`body overflow: hidden`）
- 弹窗关闭后，恢复下单页滚动
- 弹窗不影响底部吸底栏的 fixed 定位

---

## 2. 弹窗触发位置

弹窗 HTML 放置在 `<body>` 末尾、底部吸底栏之后，作为独立的浮层：

```html
<body>
  <div class="page">
    <!-- 原有下单页内容 -->
    ...
    <!-- 11. 底部吸底栏 -->
    <div class="bottom-bar">...</div>
  </div>

  <!-- 火车票引流弹窗 fdl-dialog（条件展示） -->
  <!-- AI插入 火车票引流弹窗模块（仅当满足引流条件时） -->
</body>
```

## 3. 弹窗内部结构

弹窗由以下固定部分组成，从上到下依次为：

1. **遮罩层**（`fdl-dialog`）：FDG 标准遮罩，`color-mix(in srgb, var(--color-darkgray) 60%, transparent)`，点击可关闭
2. **弹窗面板**（`fdl-dialog__panel`）：白色圆角卡片，630px 宽
3. **关闭按钮**（`fdl-dialog__close`）：右上角叉号（扩展）
4. **橙色渐变头部**（`fdl-dialog__header--gradient`）：高铁图标 + 标题 + 副标题
5. **身体**（`fdl-dialog__body`）：航线信息区 + 价格对比区
6. **脚部**（`fdl-dialog__footer`）：操作按钮（狠心离开 + 立即抢订）

### 可复制 HTML 骨架

```html
<div class="fdl-dialog fdl-dialog--visible"
     id="fdl-dialog-train-diversion"
     aria-hidden="false"
     data-spm="train-diversion-modal">

  <div class="fdl-dialog__panel"
       role="dialog"
       aria-modal="true"
       aria-labelledby="fdl-dialog-train-title">

    <!-- 关闭按钮（扩展） -->
    <button type="button" class="fdl-dialog__close" aria-label="关闭弹窗" data-fdl-dialog-close></button>

    <!-- 头部：橙色渐变 -->
    <header class="fdl-dialog__header--gradient">
      <!-- AI替换：图标 + 标题 + 副标题 -->
    </header>

    <!-- 身体：航线 + 价格 -->
    <div class="fdl-dialog__body">
      <div class="fdl-dialog__route">
        <!-- AI替换：出发城市 + 虚线 + 到达城市 -->
      </div>
      <div class="fdl-dialog__price-section">
        <!-- AI替换：优惠标签 + 价格行 + 立省金额 -->
      </div>
    </div>

    <!-- 脚部（标准 fdl-dialog__footer） -->
    <footer class="fdl-dialog__footer">
      <div class="fdl-dialog__actions">
        <button type="button" class="fdl-dialog__btn fdl-dialog__btn--secondary" data-fdl-dialog-close>狠心离开</button>
        <button type="button" class="fdl-dialog__btn fdl-dialog__btn--primary" data-fdl-dialog-close>立即抢订</button>
      </div>
    </footer>

  </div>
</div>
```

## 4. 弹窗展示/隐藏控制

### 展示条件（全部满足才展示）

1. 交易控或交叉导购控命中
2. 机票经济舱单成人惠后价 - 高铁直达二等座最低价 > 配置阈值
3. 仅存在高铁动车直达车次（G/D/C/S）
4. 用户已勾选乘机人
5. 当天同 OD 未超过频控次数（默认 1 次/天）
6. 无更高优先级弹窗（crm 弹窗优先）

### 关闭方式

- 点击关闭按钮（右上角叉号）
- 点击遮罩层
- 点击"狠心离开"按钮

### 关闭后行为

- 关闭弹窗，恢复下单页
- 记录频控计数（当天同 OD 不再弹出）
