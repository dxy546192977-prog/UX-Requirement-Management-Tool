🔔 钉钉通知设置

配置钉钉群机器人 Webhook，新需求提交后将自动发送通知到群里。Webhook URL 仅保存在本地浏览器，不会上传到云端。

获取方式：钉钉群 → 群设置 → 智能群助手 → 添加机器人 → 自定义 → 复制 Webhook 地址。
安全设置建议选择「自定义关键词」，填入 需求。

☁️ 阿里云 OSS 配置

配置阿里云 RAM 用户的 AccessKey，用于将需求数据写入 OSS 云端存储。AccessKey 仅保存在本地浏览器，不会上传到任何服务器。

获取方式：阿里云控制台 → 右上角头像 → AccessKey 管理 → 使用 RAM 用户 AccessKey → 创建。
请确保该 RAM 用户已授权 AliyunOSSFullAccess 或 OSS 写入权限。
