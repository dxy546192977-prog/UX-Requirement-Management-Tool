# 回归测试 Prompt

## 普通页面回归

- 生成一个飞猪酒店列表页，不要大图 hero，要突出搜索效率和价格比较
- 生成一个飞猪酒店详情页，要有摘要区、房型选择区和吸底预订区
- 生成一个飞猪酒店填写订单页，顶部只有黄色品牌氛围，不要像详情页
- 生成一个飞猪演唱会频道页，要求结构不能和列表页一样
- 生成一个飞猪酒店详情页，要求头图和房型图都是真实酒店照片，不要使用占位图
- 生成一个飞猪门票详情页，要求头图是当前景点或乐园相关真实图片
- 生成一个飞猪迪士尼门票页，检查是否优先使用 Disney 相关实体词搜索图片，而不是只用 theme park 泛词

## AI 页面回归

- 生成一个飞猪 AI 酒店 IM 页面，一个提问一个回答，中间允许插订单卡
- 生成一个飞猪 AI 酒店结果页，不要做成普通列表页换个标题
- 生成一个飞猪 AI 首页，重点是推荐提问和底部输入区

## 文案回归

- 生成一个飞猪 AI 预订页面，CTA 不要出现查库存、立即查库存
- 生成一个飞猪下单页，按钮语言要更像用户产品，而不是后台系统
- 生成一个飞猪旅行页面，检查图片是否与当前业务场景一致，而不是通用风景图
- 生成一个飞猪交易页，检查普通按钮是否为黄底黑字，而预订、立即抢、支付按钮是否为红底白字

## 结构回归

- 生成一个飞猪下单页，检查是否把详情页大图错误带入
- 生成一个飞猪 AI IM 页面，检查是否错误加上头像、星标或客服感装饰
- 生成一个飞猪酒店页面，检查是否具备真实图片逻辑且未把 API key 写进代码
