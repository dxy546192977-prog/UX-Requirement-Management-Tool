- 返回 + 标题 + 更多
- 返回 + 搜索
- 标题 + 右侧图标组
- 首页型无返回导航

- 搜索框
- 日期确认条
- 片区/人数/舱位/条件 chips
- 排序切换
- 条件摘要条

### 内容卡片类

- 商品卡
- 酒店卡
- 房型卡
- 价格卡
- 权益卡
- 状态反馈卡
- 说明卡

### 表单与确认类

- 输入框
- 选择器
- 日期选择器
- 步进器
- 联系人卡
- 出行人卡
- 支付方式卡
- 价格明细卡
- 吸底确认条

## AI 页面组件

### 输入相关

- 首页大输入框
- 吸底输入组件
- 推荐提问 chips
- 工具入口 pills

### 回答相关

- 文字回答块
- 主决策卡
- 预订单卡
- 商品卡
- 地图卡
- 价格卡
- 表单卡
- CTA 卡

### 状态相关

- 思考链 loading
- 输出骨架
- 空态引导
- 错误态重试
- 继续追问入口

## 组件使用原则

- 不同场景使用不同组件组合，不要固定同一套模块顺序
- 普通页面优先强调信息效率
- AI 页面优先强调输入、回答流和下一步动作
- 下单页优先用确认型组件，不要误用详情页大图和导购模块
