# 酒店首页 — 促销三卡横条（`hotel-home-promo-row`）

## 用途

**750×116** 横条，`padding` 左右 **18**、`gap` **18**；三卡：大卡 **274**、小卡 **202**、圆角 **24**；**两种 SKU 排布**：

1. **`example.html`**：左 **品牌闪促** 大卡 **APNG** 满铺；中 **特价酒店** 文案+装饰；右 **爆款直播** 文案+装饰。  
2. **`example-layout-b.html`**：左小 **品牌闪促**；中 **特价酒店** 整图底+文案；右小 **爆款直播**。

## 设计基准

- `viewport` **`width=750, user-scalable=no`**
