# 酒店首页 — 频道底栏 TabBar（`hotel-home-tabbar`）

## 用途

**酒店频道内** 固定底栏 **750×145**；五入口：**推荐**（**78×78** 黄圆 SVG，无单独文字标签）、我的收藏、酒店套餐、权益中心、我的订单；顶 **12px** 内边距；图标 **48** + 文案 **22px**。

> **非** App 全局 `tabs.*` 五大 Tab；与机票/火车「行业底栏」同类频道内导航。

## 设计基准

- 背景 **#FFFFFF**，顶阴影；`backdrop-filter: blur(10px)`  
- `viewport` **`width=750, user-scalable=no`**
