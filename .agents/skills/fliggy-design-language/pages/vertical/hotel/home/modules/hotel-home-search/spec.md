# 酒店首页 — 搜索模块（`hotel-home-search`）

## 用途

圆角白卡 **24px**、轻阴影；顶 **四 Tab**（国内/国际/民宿/钟点房，图+激活态垫高）；**表单**（城市/关键词/定位、入住离店+晚数胶囊、间数成人儿童+价格星级）；**热搜**横滑+右侧渐变；**#FFE033** 全宽圆角「搜索酒店」；底 **保障条** 图片。

## 设计基准

- 卡片宽 **750**；数字日期等 **Fliggy Sans 102**（CDN 见 `example.html` `font-face`）  
- `viewport` **`width=750, user-scalable=no`**
