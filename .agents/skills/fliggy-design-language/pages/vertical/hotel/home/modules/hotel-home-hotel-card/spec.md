# 酒店首页 — 猜你喜欢 · 酒店卡（`hotel-home-hotel-card`）

## 用途

单列/双列流中的 **348×522** 酒店卡：上 **348×348** 主图+品牌角标+距离胶囊；下信息区 **324** 宽 — 标题+星级标签、评分+短评、服务标签（双色块/简单框）、**¥694 起** + 划线价。

## 设计基准

- 卡片 `margin-left: 8px`（双列栅格时与稿对齐）  
- 价格 **SF Pro Text** 粗体 **#FF5533**；评分 **#6666FF**  
- `viewport` **`width=750, user-scalable=no`**
