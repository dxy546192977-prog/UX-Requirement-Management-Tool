# 酒店列表 — 地图模式场景（`hotel-list-map-scene`）

## 用途

**750×1304** 地图底图预览：**左上悬浮罗盘**（白圆阴影）、**距 POI 半透明顶条**、**圆角白卡** 行式酒店信息（文案与列表卡略有差异，如「君澜会员价」、标签组合）。

## 设计基准

- `viewport` **`width=750, user-scalable=no`**  
- 定位条 **rgba(0,0,0,0.4)** + `backdrop-filter: blur(15px)`
