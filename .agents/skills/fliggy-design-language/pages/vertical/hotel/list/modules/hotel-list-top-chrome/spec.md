# 酒店列表 — 顶栏与吸顶筛（`hotel-list-top-chrome`）

## 用途

**750×320** 切片：状态栏 + 返回 + **黄框搜索**（城市、住/离、间/人、占位文案）+ **地图/列表切换** + **更多**；**主筛四**（推荐排序可隐藏）；**快筛横滑**。  
点击切换图标：地图钉 ↔ 三线列表；**隐藏「推荐排序」** 且主筛 `justify-content: space-around`。

## 设计基准

- `viewport` **`width=750, user-scalable=no`**  
- 搜索框描边 **#FFE033**，主文字 **#0F131A**
