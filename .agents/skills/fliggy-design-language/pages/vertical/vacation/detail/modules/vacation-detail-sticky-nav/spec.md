# vacation-detail-sticky-nav · 商详吸顶导航

## 用途

门票/玩乐商详页 **固定顶栏**：叠在头图之上；**滚动 `≥180px`** 切换为白底 + 深色图标。

## 尺寸（@2x）

| 属性 | 值 |
|------|-----|
| 宽 | `750px` |
| 高 | **`176px`**（状态栏 `88` + 导航行 `88`） |
| 定位 | `fixed; top: 0; left: 50%; transform: translateX(-50%);` |
| z-index | 高于头图（如 `20`） |

## 状态

| | 初始（滚距 `<180`） | 滚后（`≥180`） |
|--|-------------------|----------------|
| 背景 | `transparent` 或顶区渐变遮罩 | `#FFFFFF` |
| 图标/字 | `#FFFFFF` | `#0F131A` |

## 结构

- **左**：返回
- **右**：分享、购物车/收藏、更多（间距约 `30px`，触控 `48×48`）

## 与独立 demo 的差异

独立 `example.html` 可用 **300px** 渐变区模拟沉浸预览；**整页** `example-full.html` 须与框架一致 **176px**。
