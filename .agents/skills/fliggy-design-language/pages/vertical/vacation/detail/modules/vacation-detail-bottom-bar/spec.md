# vacation-detail-bottom-bar · 商详底栏

## 用途

固定底部：**咨询 / 店铺 / 收藏** + **加入购物车**（`#FFB300`）与 **立即购买**（`#FF6200`）胶囊双按钮。

## 尺寸（@2x）

| 属性 | 值 |
|------|-----|
| 总宽 | `750px` |
| 总高 | **`173px`**（分割线 `1` + 主区 `108` + 安全区 `64`） |
| 定位 | `fixed; bottom: 0;` 水平居中 |

## 主操作区

- 左：三列图文 icon `42×42`，标签 `20px` `#5C5F66`。
- 右：按钮组总宽约 `400px`，高 `84px`，字号 `30px` 白字；左钮左圆角 `42px`，右钮右圆角 `42px`。
