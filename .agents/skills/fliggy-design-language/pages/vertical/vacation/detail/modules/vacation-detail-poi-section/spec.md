# vacation-detail-poi-section · POI 信息区

## 用途

白卡内连贯模块：**所属景区**（封面 + 渐变信息区 + 右 peek 图）、**该景区还在卖** 标题、横向类目 tab、双列商品卡。

## 容器（@2x）

| 属性 | 值 |
|------|-----|
| 宽 | `714px` |
| 圆角 | `12px` |
| 溢出 | `hidden`（横向 peek / 滚动在卡内裁切） |

## Tab

- 选中：`#FFF7CC`，右下角黄角标 + 对勾可选。
- 未选：`#F5F6F7`；促销 tab 可带顶横幅 `#FF8C1A`。

## 商品小卡

- 图 `130×130`，圆角 `12px`；价格 `#FF5533`；月销弱色 `#919499`。
