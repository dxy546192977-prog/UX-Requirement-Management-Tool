# vacation-detail-basic-info · 商详头部基础信息

## 用途

头图后首张白卡：价格、优惠券、品牌、标题（最多两行）、服务保障条、评价摘要、榜单入口。

## 容器（@2x）

| 属性 | 值 |
|------|-----|
| 宽 | `714px` |
| 内边距 | `24px` |
| 模块竖间距 | `18px` |
| 圆角 | `12px` |

## 要点

- 主价格色 `#FF5533`；删除价/弱文案 `#919499`。
- 评分徽章可用紫渐变（`#6666FF` → `#9999FF`），**小面积**。
- 保障条：左侧黄标 + 右侧米色渐变条 + 右箭头。
- 榜单条：浅米色渐变底，文案色约 `#805540`。
