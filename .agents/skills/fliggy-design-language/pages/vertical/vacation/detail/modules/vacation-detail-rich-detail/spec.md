# vacation-detail-rich-detail · 图文详情

## 用途

长图文说明：标题「图文详情」+ 长图内容 + **展开全部**（默认截断 + 底渐变遮罩）。

## 容器（@2x）

| 属性 | 值 |
|------|-----|
| 宽 | `714px` |
| 背景 | `#FFFFFF` |
| 圆角 | `12px` |

## 交互

- 默认 `max-height` 截断内容，底部渐变浮层 + 「展开全部」按钮。
- 展开后移除高度限制并隐藏浮层（实现由业务脚本完成）。
