# vacation-detail-hero-indicator · 头图胶囊指示器

## 用途

头图下方（或叠于头图底）切换 **图集 / 特色 / 印象 / 服务** 等视图。

## 尺寸（@2x）

| 属性 | 值 |
|------|-----|
| 外容器建议宽 | `714px`（整页内水平居中，左右 `18`） |
| 外容器高 | `72px` |
| 内胶囊条高 | `48px`，`border-radius: 24px` |
| 条背景 | `rgba(0,0,0,0.3)` |
| 选中项 | 背景 `#FFE033`，字 `#0F131A` |
| 未选中 | 透明底，字 `#FFFFFF` |
