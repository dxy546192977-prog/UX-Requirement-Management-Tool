# vacation-search-group-tour-card · 跟团游卡

## 用途

列表中的 **左图右文** 线路商品卡（常规高度约 `354px` 内容区，示例为 `330px` 图高 + 内边距）。

## 容器（@2x）

| 属性 | 值 |
|------|-----|
| 宽 | `750px` |
| 背景 | `#FFFFFF` |
| 内边距 | `12px 18px` |
| 图文间距 | `18px` |

## 左图

- `226 × 330px`，圆角 `12px`；角标、底栏遮罩按设计。

## 右区

- 标题 `30px`；评分紫 `#6666FF`；亮点/标签/店铺/价格 `#FF5533` 等见 `example.html`。
