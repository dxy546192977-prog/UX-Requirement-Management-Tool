# vacation-search-dropdown-filter · 下拉筛选条

## 用途

类目下 **一行多列**：出发地、筛选、日期/天数、人数、排序等；点击展开面板由业务实现。

## 尺寸（@2x）

| 属性 | 值 |
|------|-----|
| 宽 | `750px` |
| 高 | `72px` |
| 背景 | `#FFFFFF` |
| 字 | `26px` `#0F131A` |
| 项内 | 文 + 下箭头，`gap` 约 `10px` |

## 布局

- 首列可固定宽（如 `187.5px`），其余 `flex: 1` 均分。
