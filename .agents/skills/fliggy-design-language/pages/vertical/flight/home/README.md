# 机票首页（`vertical.flight.home`）

- **框架**：[page-frame.md](page-frame.md)  
- **闭包**：[manifest.md](manifest.md)  

## 模块

| slug | 说明 |
|------|------|
| `flight-home-atmosphere` | 首屏氛围（沉浸式顶栏 + 背景 + 渐变） |
| `flight-home-mini-search` | 小搜（一级/二级 Tab + 搜索卡片） |
| `flight-home-kingkong` | 小金刚（五入口） |
| `flight-home-savings` | 省钱（三卡） |
| `flight-home-special-fares` | 特价机票（主题 Tab + 列表） |
| `flight-home-activity-strip` | 活动长图（双图 + 间距） |
| `flight-home-industry-tabbar` | 行业底栏（值机/收藏等五入口，非 App 主 Tabbar） |
