## 用途

**吸底** 排序栏：**时间排序 / 低价优先 / 惠后价**（三按钮）；总高 **160px**（上 **96** 按钮区 + **64** 安全区）。

---

## 规格摘要

| 项 | 值 |
|----|-----|
| 外层 | **750×160**，**fixed bottom**，阴影见供稿 |
| 按钮区 | **750×96**，白底，顶圆角 **24px**，`padding: 0 24px` |
| 单按钮 | 约 **234×96**，`flex-grow: 1`，图标 **36×36**，文案 **20px** |
| 选中态 | 供稿内高亮样式（见 `example.html`） |

整页对齐：`page-frame` 要求 **left:50% + translateX(-50%)** 与 **160px** 底部留白。

详参供稿 `输出组件/底bar/README.md`。
