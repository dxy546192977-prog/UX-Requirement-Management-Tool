## 用途

## 规格摘要

| 项 | 值 |
|----|-----|
| 高度 | 约 **280–340px**（以稿为准） |
| 背景 | 白到页灰 **渐变** |
| 与上下 | **紧贴导航**；下接 **提示百叶窗** |
| 资产 | 灰色单程 icon **42×42**；出行提醒 **26×26**；航司 logo **22×22**；分隔 **2×20** |

**出行提醒** 属本卡默认态，不单独模块。详参 `输出组件/行程卡/README.md`。
