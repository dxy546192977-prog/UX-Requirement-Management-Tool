# 模块：租车 LIST · 新客专享黄条

## 用途

LIST **仅**使用优惠券组件中的 **`promo-bar`** 变体（礼物 + 新客专享标 + 文案 + 渐变遮罩 + **领取**）。

## 尺寸

- 外层 **LIST 专用**：**750×96** 白底（`.list-promo-strip`），垂直居中内嵌 **714×72** `.promo-bar`（与供稿 promo 一致）。
- 与上方大筛选、下方灰列表 **无灰缝**，白顶连续感由画布叠层保证。

## 字体强调

- 「新客专享」四字：**#FF3333**；优先 **Fliggy Fonts** 斜体（供稿图为 `promo-icon-tag` 资源）。

## 资源

礼物、新客专享条标、文案与 OTA 小黄条 demo 同源，见 `example.html`。
