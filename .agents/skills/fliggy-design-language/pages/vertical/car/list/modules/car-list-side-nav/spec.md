# 模块：租车 LIST · 侧导航

## 用途

左侧 **148×920** 纵向 **8** 个分类 tab；首项「全部车型」默认选中。

## 单项

- **115px** 高；未选背景 **#F2F3F5**，选中 **#FFFFFF**
- 选中：渐变 SVG 铺满、`6×79` 左黄条、标题/价格 **底对齐**；字重 **500**，价格色 **#0F131A**
- 未选：标题/价格 **顶对齐**；标题 **400**，价格 **#919499**

## 数据

见 `manifest.md` 与供稿表格（全部车型 ¥132起 … 房车 ¥1048起）。

渐变图、黄条 SVG 见 `example.html`。
