# 租车 LIST · 设计框架与组件

生成或校对 **整页 LIST** 时以本文件与 `manifest.md` 为准：复用 `modules/` 内切片 DOM 与视觉，**仅可**改文案、显隐、顺序与少量拼装；**禁止**新造卡片结构。

## 1. 固定层级顺序

```text
上状态栏 → 二级导航 → 快筛栏 → 大筛选 → 新客专享黄条 → 主列表（侧导航 + 车型卡）
```

## 2. 组件映射（本仓库）

| 区域 | 模块目录 |

| 状态栏 | `car-list-status-bar/` |
| 二级导航 | `car-list-nav/` |
| 快筛栏 | `car-list-quick-filter/` |
| 大筛选 | `car-list-big-filter/` |

## 3. 显隐与 LIST 特化

- 快筛栏：**不展示**「联订任务」；保留横向超宽，**右侧尾项被裁切**暗示可滑。
- 大筛选：三张整卡 + 右侧装饰与轨道溢出。
- 黄条：**仅** `promo-bar`（新客专享），**不**用双条、**不**用 88VIP。
- 二级导航：右上**仅更多**；**不**出现收藏/分享；中间**不**单独露「杭州-苏州」类双城标题（以取还块文案为准）。
- 车型卡：**不**展示划线价、会员价、月销、展开第二报价、联订描边、展开更多行等隐藏分支。

## 4. 强识别

- 搜索框：**182×72**，白底 **#FFE033** 边框，占位 **搜车型**。
- 「新客专享」标识：**#FF3333**；优先使用供稿 **飞猪体斜体**（`Fliggy Fonts`），缺字重时回退斜体 PingFang。
- 侧导航选中项：**6×79** 左侧黄条 `#FFDD00` + 渐变遮罩资源图。
- 右列首张：**大众朗逸** + **最近浏览** 角标。
