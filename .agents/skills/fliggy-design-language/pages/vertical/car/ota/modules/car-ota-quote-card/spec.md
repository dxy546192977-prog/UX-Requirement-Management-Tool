# 模块：租车 OTA · 报价卡片

## 用途

报价列表中的 **714px** 宽白卡；多种变体组合见 `design-framework-components.md` 与 `manifest.md`。

## 通用

- 容器：宽 **714px**，背景 `#FFFFFF`，圆角 **12px**
- 标题：**30px** 字重 **500** `#0F131A`（一口价为金棕 `#805540`）
- Tab 区（若有）：高 **90px**，Tab **333×66**，轨道 `#F2F3F5`，激活项内白底
- 内容区内边距：上 **12** 右 **24** 下 **24** 左 **18**，子项间距 **12**
- 订购按钮：**84×84**，`#FF8C1A`，白字 **订** **36px**

## 变体一览（8 张报价卡）

| 标题 | 结构要点 |
|------|----------|
| 品牌周·精选车 | 无 Tab；信息行 + 标签 + 商家 + 报价 |
| 总价最低·特惠车 | 双 Tab 左激活 |
| 品质新车 | 双 Tab 左激活；**飞猪优选** 徽章在商家行前 |
| 飞猪优选车（送取） | 无 Tab；三行信息，末行灰色「可选：收费上门送取车」 |
| 飞猪优选车（取还） | 无 Tab；车龄 + 取· + 还· 三行 |
| 全国连锁·大牌好车 | 双 Tab 左激活；**全国连锁** 徽章 |
| 小伤免赔·剐蹭无忧车 | 标题行右侧 **18×18** 箭头 |
| 一口价·舒心租车 | 金棕标题 + 箭头；报价区；底部 **6** 项权益栏 |
