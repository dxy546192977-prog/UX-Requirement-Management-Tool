# 模块：租车 OTA · 租车取还信息

## 用途

摘要区底部：地点 + 取还时间 + 租期胶囊；顶圆角融白卡，底无圆角融页面灰底。

## 尺寸

- **750×116**
- 内边距：上下 18px，左 42px，右 24px；行间距 12px
- 背景：`linear-gradient(180deg, #FFFFFF 50%, #F2F3F5 100%)`
- 圆角：**仅顶部** `24px 24px 0 0`

## 变体

- **同地**：单行地点
- **异地**：地点A + 异地图标 + 地点B（图标见 example）

## 时间行

取车时间、`2天` 胶囊（边框 `#D2D4D9`，`border-radius: 30px`，字 20px `#5C5F66`）、还车时间。
