# 我的 Tab（`tabs.mine`）

- **框架**：[page-frame.md](page-frame.md)  
- **闭包**：[manifest.md](manifest.md)  

## 模块

| 目录 | 说明 |
|------|------|
| `mine-profile-header` | 用户信息、会员标签、进度、客服/设置、紫色会员卡、省钱卡 |
| `mine-toolbar` | 订单行 / 收藏与数据行（数字徽章）/ 常用工具行 + 指示器 |
| `mine-feeds` | 双列推荐 + 左列顶部轮播大卡 |

底栏：**tabbar**（「我的」为当前态）。
