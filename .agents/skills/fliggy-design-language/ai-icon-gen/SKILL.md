# AI Icon Prompt 生成器

## 核心定位

将用户给出的抽象主题/命题，转化为可直接喂给 AI 绘图工具（Midjourney、DALL-E、Stable Diffusion、GPT-4o Image 等）的高质量 icon prompt。

## 工作流程

### Step 1: 收集需求

使用 AskUserQuestion 向用户了解以下信息（一次性询问，不要拆开问）：

1. **视觉风格**（必选）：
   - 3D立体（C4D/OC/Blender风格，圆润有质感）
   - 磨砂玻璃拟态（frosted glass，半透明通透感）
   - 扁平化（flat design，简洁色块）
   - 黏土/Clay（clay render，软萌手工感）
   - 等轴测（isometric，45度微缩世界）
   - 写实渲染（photorealistic，照片级真实）
   - 新拟物（neumorphism，柔和凸起质感）
   - 线性图标（line art，简洁线条）
   - 像素风（pixel art，复古游戏感）

2. **配色倾向**（必选）：
   - 蓝紫科技感
   - 多彩低饱和（马卡龙色）
   - 黑白金（高端奢华）
   - 多彩渐变（鲜艳活力）
   - 莫兰迪色（高级灰调）
   - 跟随品牌色（用户指定）

3. **使用场景**（可选，有助于调整细节）：
   - App/网页UI
   - 营销/运营页面
   - 文档/PPT
   - 社交媒体

### Step 2: 理解命题，拆解主体

收到用户的命题后，做以下思考（内部过程，不必完整展示给用户）：

1. **提炼核心概念**：这个命题的核心动作/对象是什么？
2. **转化为具象元素**：用什么实体物件来象征这个概念？选择1-2个主要元素 + 0-1个辅助元素。
3. **确定动态/关系**：元素之间有什么互动关系？

**注意**：主体元素的选择应该是直觉性的、一看就懂的。避免过于抽象或需要解释的隐喻。

示例拆解：
- "签约入驻" → 合同文件 + 签字笔（笔尖触碰文件） + 认证印章
- "数据分析" → 柱状图/饼图 + 放大镜 + 上升趋势箭头
- "云存储" → 云朵 + 上传箭头 + 文件/文件夹
- "在线支付" → 手机/银行卡 + 金币 + 盾牌（安全）
- "团队协作" → 拼图拼合 + 多人头像 + 连接线

### Step 3: 组装 Prompt

#### 核心公式

```
主体描述（含元素互动）, 风格词, 材质词, 色彩词, 灯光词, 视角词, 渲染词, 背景词, 平台参考词, 分辨率/画质词
```

#### 关键原则

1. **关键词串式表达**：用逗号分隔的短关键词，不要写叙事长句
2. **主体描述尽量精炼**：用15-30个字描述主体和关键细节即可
3. **每个维度命中1-2个关键词**：不要在同一维度堆砌太多同义词
4. **维度覆盖完整**：确保风格、材质、灯光、视角、渲染、背景、画质都有涉及
5. **中文prompt默认输出**：除非用户要求英文

#### 各维度关键词速查

详细关键词库见 [keywords-reference.md](keywords-reference.md)，以下是按风格的常用组合：

### Step 4: 输出格式

按以下格式输出每个命题的 prompt，以便用户直接复制使用：

```
**[命题名称]**

> [完整prompt内容]
```

如果是批量多个命题，保持所有prompt的风格关键词一致，仅替换主体描述部分，确保系列图标风格统一。

## 进阶技巧

### 同一命题的多方案输出

如果用户对某个命题不确定，可以提供2-3个不同的主体元素方案：

```
**数据分析 - 方案A（图表为主）**
> 柱状图与上升趋势箭头图标, ...

**数据分析 - 方案B（洞察为主）**
> 放大镜聚焦数据面板图标, ...
```

### 主体元素的组合层次

- **单元素**：适合极简风格，一个核心物件即可（如：一把锁 = 安全）
- **双元素互动**：最常用，两个元素产生关系（如：钥匙插入锁 = 解锁/认证）
- **三元素组合**：信息量更丰富但不要过载（如：盾牌+锁+勾 = 安全认证通过）

### 风格一致性要点

批量生成图标时，以下关键词必须在所有prompt中保持一致：
- 风格词（如3D艺术, C4D）
- 材质词（如磨砂玻璃质感）
- 灯光词（如soft lighting）
- 视角词（如等轴测视角）
- 渲染词（如OC渲染器, Blender渲染）
- 背景词（如干净白色背景）
- 画质词（如16K分辨率）

仅主体描述和色彩细节可以变化。

## 注意事项

1. **不要过度描述**：icon是小尺寸视觉元素，主体要简洁明了，一眼可辨
2. **避免堆砌同义词**：每个维度选最精准的1-2个词即可
3. **色彩要具体**：说"蓝紫色渐变"比说"科技感的颜色"更有效
4. **保持中文为主**：除了专有名词（C4D、Blender、Octane等），尽量用中文表达
5. **Midjourney参数**：如果用户使用Midjourney，可在prompt末尾追加参数如 `--s 250 --v 6.0`
