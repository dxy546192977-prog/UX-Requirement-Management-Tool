## 十一、Midjourney 专用参数

| 参数 | 语法 | 说明 |
|------|------|------|
| 风格化 | `--s 250` | 控制AI审美强度，100为默认，250偏艺术 |
| 版本 | `--v 7` 或 `--v 6.0` | 模型版本 |
| 宽高比 | `--ar 1:1` | 图标通常用1:1 |
| 排除 | `--no text, watermark` | 排除不需要的元素 |
| 混乱度 | `--c 10` | 4张图之间差异度 |
| 质量 | `--q 2` | 更高细节 |
