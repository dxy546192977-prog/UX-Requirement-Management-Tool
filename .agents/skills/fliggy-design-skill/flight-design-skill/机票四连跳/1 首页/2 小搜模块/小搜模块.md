# 小搜模块 (SkillSearch)

## 这是什么？

这是机票首页的核心搜索组件，也是用户使用频率最高的模块。你可以把它想象成一个功能完整的"机票搜索表单"，用户在这里输入出发地、目的地、日期等信息，然后点击搜索按钮查找航班。

它由以下几个部分组成：
*   **一级导航 Tab**：顶部的交通类型切换栏，包含"机票"、"特价机票"、"火车票"、"汽车票"、"接送机"五个品类，当前默认选中"机票"。左侧有飞猪品牌 Logo 区域，选中态下方有黄色渐变指示条。
*   **二级导航 Tab**：行程类型切换，包含"单程"、"往返"、"多程"三个选项，选中态文字加粗并有底部指示条。
*   **搜索卡片**：白色圆角卡片，包含以下输入行：
    *   **出发/到达城市行**：左右两个城市名 + 中间的飞机换向图标（68×68px 切图）。
    *   **日期行**：出发日期，数字使用 Fliggy Sans 102 字体（48px），旁边显示"月"和"周几"。
    *   **乘机人行**：显示乘机人数量和提示文本，右侧可能有引导气泡。
    *   **舱位切换**：经济舱/公务舱/头等舱的胶囊切换器。
*   **搜索按钮**：醒目的黄色（`#ffe033`）大按钮，高 84px，圆角 42px，文字"搜索机票"。按钮右上角可能有红色促销角标。
*   **搜索历史**：按钮下方的最近搜索记录，可清除。

## 如何配置它？

### 一级导航 Tab

一级导航使用切图实现，包含 5 个交通品类。选中态下方有黄色渐变指示条。

*   **当前选中品类 (`activeCategory`)**: 默认选中"机票"。
    *   可选值：`'机票'` | `'特价机票'` | `'火车票'` | `'汽车票'` | `'接送机'`

当前预设的品类列表：

| 序号 | 名称 | 图标文件 |
|------|------|------|
| 1 | 机票 | `./assets/机票.svg` (72×34px) |
| 2 | 特价机票 | `./assets/特价机票.svg` (136×34px) |
| 3 | 火车票 | `./assets/火车票.svg` (78×27px) |
| 4 | 汽车票 | `./assets/汽车票.svg` (80×27px) |
| 5 | 接送机 | `./assets/接送机.svg` (122×27px) |

### 二级导航 Tab

*   **行程类型 (`tripType`)**: 当前选中的行程类型，选中态文字 30px/500 深色 + 底部 46×6px 指示条，未选中态文字 30px/500 浅灰色。
    *   可选值：`'单程'` | `'往返'` | `'多程'`
    *   例如：`tripType: '单程'`

### 搜索卡片

#### 出发/到达城市

*   **出发城市 (`departCity`)**: 左侧显示的出发城市名，42px/500。
    *   例如：`departCity: '杭州'`

*   **到达城市 (`arriveCity`)**: 右侧显示的到达城市名，42px/500，右对齐。
    *   例如：`arriveCity: '北京'`

*   **换向按钮**: 中间的飞机图标（68×68px 切图），点击可交换出发和到达城市。

#### 日期

*   **出发日期 (`departDate`)**: 日期数字使用 Fliggy Sans 102 字体（48px），旁边显示"月"和"周几"（26px PingFang SC）。
    *   例如：`departDate: { month: 7, day: 15, weekday: '周二' }`

#### 乘机人

*   **乘机人标题 (`paxTitle`)**: 默认显示"1位成人"。
    *   例如：`paxTitle: '1位成人'`

*   **提示文本 (`paxTip`)**: 绿色提示文字（`#149aa8`），如"快速找到低价票"。
    *   例如：`paxTip: '快速找到低价票'`

*   **引导气泡 (`bubble`)**: 半透明黑底气泡，左侧有三角箭头指向乘机人区域，内含文案和关闭按钮。
    *   例如：`bubble: { text: '添加乘机人享更多优惠', visible: true }`

#### 舱位

*   **舱位选项 (`cabinOptions`)**: 胶囊切换器，选中态白底 + 阴影，未选中态透明。
    *   可选值：`'经济舱'` | `'公务舱'` | `'头等舱'`
    *   例如：`cabinOptions: ['经济舱', '公务舱', '头等舱'], activeCabin: '经济舱'`

### 搜索按钮

*   **按钮文案 (`searchText`)**: 搜索按钮上的文字。
    *   例如：`searchText: '搜索机票'`

*   **促销角标 (`promoTag`)**: 按钮右上角的红色渐变角标，显示促销信息。不对称圆角设计。
    *   例如：`promoTag: '立减¥50'`

### 搜索历史

*   **历史记录 (`history`)**: 最近的搜索记录列表，显示在按钮下方，右侧有"清除"按钮。
    *   例如：`history: ['杭州→北京', '上海→广州']`

## 一些例子

### 1. 修改出发和到达城市

替换城市名文字即可：

```html
<span class="skill-search__city">上海</span>
<!-- 中间飞机图标 -->
<span class="skill-search__city skill-search__city--right">广州</span>
```

### 2. 修改出发日期

替换日期数字和周几文字：

```html
<div class="skill-search__date">
  <div class="skill-search__date-nums">
    <div class="skill-search__date-num">
      <span class="skill-search__date-num-text">8</span>
    </div>
    <div class="skill-search__date-unit">
      <span class="skill-search__date-unit-text">月</span>
    </div>
    <div class="skill-search__date-day">
      <span class="skill-search__date-day-text">15</span>
    </div>
  </div>
  <div class="skill-search__date-weekday">
    <span class="skill-search__weekday">周五</span>
  </div>
</div>
```

### 3. 修改促销角标文案

替换角标内的文字：

```html
<span class="skill-search__promo-btn">暑期特惠</span>
```

### 4. 切换舱位选中态

将 `--active` 类移到目标舱位选项上：

```html
<div class="skill-search__cabin">
  <button class="skill-search__cabin-item skill-search__cabin-item--normal">经济舱</button>
  <button class="skill-search__cabin-item skill-search__cabin-item--active">公务舱</button>
  <button class="skill-search__cabin-item skill-search__cabin-item--normal">头等舱</button>
</div>
```

## 设计细节备忘

*   **一级导航毛玻璃**：一级 Tab 背景使用 `backdrop-filter: blur(12px)` + 半透明渐变 + 60% 白色边框，营造毛玻璃质感。
*   **黄色指示条**：选中品类下方的指示条使用黄色渐变（`#ffe033` → 透明），尺寸 84×24px，圆角 24px。
*   **日期字体**：日期数字必须使用 Fliggy Sans 102 字体（48px），"月"和"周几"使用 PingFang SC（26px），两者基线对齐。
*   **促销角标圆角**：角标使用不对称圆角（左 16px / 上 19px / 右上 19px / 右下 2px），形成独特的"飘带"造型。
*   **气泡箭头**：引导气泡左侧的三角箭头使用 CSS border 实现（8×18px），指向左侧。
*   **搜索按钮阴影**：整个搜索卡片有 `0 6px 20px rgba(0,0,0,0.03)` 的轻微阴影。
