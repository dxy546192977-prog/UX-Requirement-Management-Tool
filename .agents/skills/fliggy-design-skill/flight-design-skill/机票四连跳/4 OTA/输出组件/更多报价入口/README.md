# 更多报价入口

用于商品模块底部的收尾入口，承载“更多报价 ¥1043起”和展开箭头。

## 用途

- 作为当前页商品列表的收尾入口
- 连接更多报价的下一级页面或展开态

## 默认可见态

- 左侧：`更多报价 ¥1043起`
- 右侧：`展开` + 向下箭头

## 结构层级

```html
<section class="more-entry">
  <div class="more-copy"></div>
  <div class="more-action"></div>
</section>
```

## 关键尺寸 / 字体 / 排版

- 容器高度：`78px`
- 主文案：`26px`，`500`
- 操作区：`24px`
- 箭头：右箭头资源旋转 `90deg`

| 名称 | 用途 | URL |

## 可选态

- 展开后文案变化
- 箭头旋转为收起态

## 组合约束

- 位于商品模块内部，4 张商品卡之后
- 不单独漂浮在商品模块外部
