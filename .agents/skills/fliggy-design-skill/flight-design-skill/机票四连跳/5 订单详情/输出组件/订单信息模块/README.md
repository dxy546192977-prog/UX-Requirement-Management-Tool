# 订单信息模块

用于展示订单完成后的基础信息与权益说明，通常位于出行信息之后，包括下单时间、里程奖励、供应商与支付单号。

**AI 可配置项**：修改标题 / 配置下单时间 / 控制是否展示里程激励行 / 修改供应商名称 / 修改支付单号 / 修改领取按钮文案

## 模块结构

### 1. 标题头 `.order-info-header`

| 属性 | 值 |
|------|------|
| 高度 | 80px |
| 内边距 | 24px |
| 背景色 | var(--color-white) |
| 标题字号 | 28px / 500 |

### 2. 信息列表 `.order-info-list`

列表采用连续白底行，每行左右两列。

| 属性 | 值 |
|------|------|
| 下内边距 | 30px |
| 行间距 | 20px ~ 30px |
| 标签样式 | 24px / 400 / var(--color-midgray) |
| 值样式 | 24px / 400 / var(--color-darkgray) |

### 3. 里程奖励行 `.mileage-row`

| 属性 | 值 |
|------|------|
| 左侧文案 | “订单完成，可领取 XXX 飞猪里程” |
| 图标 | 24 x 24px |
| 右侧按钮 | 108 x 48px 黄色胶囊 |
| 按钮文字 | 24px / 500 / var(--color-darkgray) |

完整 HTML 结构详见 [example.html](example.html)。
