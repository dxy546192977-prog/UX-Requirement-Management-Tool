## 用途

**功能入口矩阵**：2 行 × 4 列共 8 个图标入口，覆盖机票核心功能快捷入口。白底圆角卡片，每个入口由图标 + 主标题 + 可选副标题组成。

---

## 规格摘要

| 项 | 值 |
|----|-----|
| 容器 `.flight-kingkong` | **702×240**，白底 `var(--color-white)`，圆角 **var(--radius-l)** |
| 内边距 | **24px 18px** |
| 网格 | `display: grid; grid-template-columns: repeat(4, 1fr); gap: 18px 0` |
| 图标 | **60×60**，圆角 **var(--radius-m)**，居中 |
| 主标题 | **24px/500 var(--color-darkgray)**，居中，`margin-top: 8px` |
| 副标题 | **20px/400 var(--color-pay-1)**，居中（如「低至¥99」），可选 |

## 状态与变体

| 变体 | 说明 |
|------|------|
| **默认 2×4** | 8 个入口，两行四列 |
| **扩展 2×5** | 10 个入口，两行五列；容器宽度不变，图标间距自适应 |
| **带角标** | 个别入口右上角红点或「NEW」标签（走 `badge-dot` 组件） |
